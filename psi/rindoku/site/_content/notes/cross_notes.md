## 1. 全章で一貫していること

| 一貫する軸 | 現れ方 |
|---|---|
| **「骨格 A^T C A」フレーム** | Intro で提示 → Ch1（差分行列）→ Ch2（ばね・回路・最小二乗）→ Ch3（連続BVP・FEM）→ Ch8（最適化のKKT）まで、問題が変わっても同じ組立図に回収される |
| **「構造が解法を決める」** | Ch1（疎・帯・対称 → 解法選択）→ Ch7（帯幅・フィルイン → 計算量）→ Ch8（鞍点構造 → KKT行列）。"何を解くか"より"どんな形をしているか"が先 |

| **周波数分解という共通言語** | Ch4（フーリエ級数）→ Ch5（複素平面・特異点）→ Ch6（分散関係 ω(k)・増幅率 G(k)）。空間/時間の議論をスペクトル側に翻訳して統一する |
| **"説明責任"への意識** | 全章の「自分にどう効くか」に共通して、「言語化できる」「説明が通る」「原因を構造として切り分けられる」が繰り返し登場する |

## 2. 章ごとに変化すること

| 軸 | 前半（Intro–Ch3） | 転換点（Ch4–Ch5） | 後半（Ch6–Ch8） |
|---|---|---|---|
| **対象** | 離散（行列）→ 連続（微分方程式）へ拡張 | 表現の変換（時間/空間 → 周波数/複素平面） | 時間発展・大規模計算・最適化へ |
| **問いの型** | 「Ku=f をどう組むか」（静的平衡） | 「関数をどう分解・近似するか」（表現論） | 「どう解くか」「何を最小にするか」（計算戦略・設計） |
| **難しさの源泉** | 境界条件・零空間・特異性 | 不連続（Gibbs）・特異点の位置 | 安定性（|G|>1）・スケーリング（N^4）・不適切性 |
| **"壊れ方"の種類** | 解が一意に決まらない / 不存在 | 近似が振動する / 収束が遅い | 計算が爆発する / 解が暴れる |
| **設計者の仕事** | AとCを見つけてKを組む | 基底・表現を選ぶ | 順序付け・前処理・正則化を選ぶ |

## 3. 著者の全体仮説（読み取り）

1. **計算科学の本質は「構造の発見と利用」である**
   - 問題の数理構造（疎性・対称性・解析性・周期性…）を見抜くことが、解法選択・計算量・精度のすべてを決める。計算機パワーの前に構造の理解がある。

2. **分野横断の統一言語は線形代数である**
   - ばね、回路、流体、最適化、信号…すべてが A^T C A → Ku=f → 部分空間の射影 に帰着する。分野固有の用語の下に共通の骨格がある。

3. **「単純化してから戻す」が唯一のスケーラブルな学習法**
   - 現実の複雑さに正面突入せず、段階的に仮定を外していく。この姿勢自体が方法論の提案。

## 5. 怪しい点・留保が必要な点

| # | 内容 | 章 | 疑問・留保 |
|---|---|---|---|
| 1 | 「単純化をやめて複雑さへ戻すタイミング」の判断基準 | Intro | 後続章に委ねられているが、結局は明示的な基準が出てこない。実務では誤差許容・検証コストとのトレードオフで決まるはずだが、そこは読者任せ |
| 2 | A^T C A フレームの適用限界 | Ch2–Ch3 | 初見の系で「何を u とするか」「A をどう切るか」は慣れ＝暗黙知に依存する。フレーム自体は強力だが、適用の入口にハードルがある |
| 3 | Gibbs対策の実務指針 | Ch4 | フィルタ・窓関数・スペクトル粘性などの選択基準が章内では不足。「現象→対策→副作用」のマップが欲しい |
| 4 | 複素平面での特異点推定 | Ch5 | 「特異点の位置が精度を決める」は美しいが、実データからその位置をどう同定するかは未整理 |
| 5 | 直接法→反復法の切替規模の定量比較 | Ch7 | 「N が大きいと反復法」は分かるが、具体的なクロスオーバー（問題構造×規模×ハードウェア）は抜粋では追えない |
| 6 | 非凸・大規模最適化での正則化・双対の選択 | Ch8 | 凸の範囲では統一的だが、実務で頻出する非凸ケースへの拡張指針が抽象的 |

## 付録：章ごとの「壊れ方」と「直し方」一覧

| 章 | 典型的な壊れ方 | 原因の構造 | 直し方の方向 |
|---|---|---|---|
| Ch1 | 解が一意に決まらない | 零空間が非自明（境界条件不足） | 拘束を追加 / 境界条件を変更 |
| Ch3 | FEMの結果がおかしい | 境界条件の入れ方（固定/自由）が間違い | w の連続性・拘束条件を点検 |
| Ch4 | 近似が振動する（Gibbs） | 不連続 → 係数減衰が遅い（1/k） | フィルタ / 窓関数 / 別の基底 |
| Ch5 | 多項式近似が端で暴れる | 等間隔点 + Runge現象 | チェビシェフ点 + バリセントリック |
| Ch6 | 数値解が発散する | |G(k)| > 1（離散化の不安定） | Δt/Δx の制約を満たす / スキーム変更 |
| Ch7 | 疎行列なのにメモリ不足 | 帯幅大 → フィルイン爆発 | リオーダリング / 反復法へ切替 |
| Ch8 | 逆問題の解が暴れる | 不適切問題（ill-posed） | 正則化（ペナルティ / Tikhonov） |
