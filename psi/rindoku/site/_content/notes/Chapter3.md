# Chapter 3 — Boundary Value Problems

## 要点
- 境界値問題 \(-\frac{d}{dx}(cu')=f\) を \(K=A^TCA\) と同じ骨格で捉え、弱形式→有限要素法（FEM）へつなげる章
- 差分の \(A\) が微分演算子 \(d/dx\) に置き換わっても同じフレームで「解ける形」に落とす

## この章の主張
- 連続でも骨格は \(A^TCA\) — ひずみ/勾配→構成則→力のつり合いを押さえると微分方程式は同じ設計図の延長
- 境界条件は仕様であり解法の一部 — 固定（\(u=0\)）と自由（\(w=0\)）の選択が解の意味と実装を決める
- 弱形式がFEMへの"橋" — 任意の試験関数 \(v\) に対する積分式にすると近似空間を設計できる

## 重要キーワード
- 境界値問題（BVP）
- 弱形式（weak / variational form）
- 有限要素法（FEM）
- \(A^TCA\) フレーム（勾配・構成則・つり合い）
- 境界条件（固定/自由）

## 自分メモ
- 初動：「未知量 \(u\)」「内部量 \(w=cu'\)」「境界で拘束するのは \(u\) か \(w\) か」をまず決める
- 弱形式→FEMで組むと、係数変更・境界追加・形状変更に強い実装になる
- 結果が変なら拘束条件（固定/自由）と \(w\) の連続性を点検
- 差分とFEMの使い分け（精度・実装コスト・境界の複雑さ）は別途整理が要る
