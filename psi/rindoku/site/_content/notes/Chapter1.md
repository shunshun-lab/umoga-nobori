# Chapter 1 — Applied Linear Algebra

## この章の核
行列を「作用素」として捉え、4つの代表行列族 $K_n, C_n, T_n, B_n$ を通じて、**構造（疎・帯・対称・正定値・境界条件）が解法・安定性・計算量をすべて決める**ことを示す。差分方程式と微分方程式の対応を軸に、消去法（$LDL^T$）→逆行列/Green関数→固有値/固有ベクトル→正定値性→数値線形代数（$LU, QR, SVD$）まで一気通貫で展開する。

---

## 定義

- **正定値（positive definite）**: 対称行列 $S$ について $u^T S u > 0$ （全 $u \neq 0$）
  - 同値条件: 全ピボット正 / 全固有値正 / 全首座小行列式正 / $K = A^T A$（$A$ の列が独立）/ Cholesky分解 $A = \mathrm{chol}(K)$ が存在
- **半正定値（positive semidefinite）**: $u^T S u \ge 0$ （全 $u$、零空間が非自明でもよい）
- **条件数（condition number）**: $c(K) = \lambda_{\max} / \lambda_{\min}$（正定値 $K$）、一般には $c(A) = \|A\|\|A^{-1}\| = \sigma_{\max} / \sigma_{\min}$
- **Green関数**: $-u'' = \delta(x-a)$ の解 $u(x,a)$。離散版は $(K^{-1})_{ij}$ = 点 $j$ の荷重に対する点 $i$ の応答
- **擬似逆行列**: $A^+ = V\Sigma^+ U^T$（$\Sigma^+$ は $\sigma_i \to 1/\sigma_i$（$i \le r$）、$0$（$i > r$））

| $K_n$ | 固定-固定 2階差分行列（対角2、副対角-1） | $\begin{bmatrix}2&-1&0\\-1&2&-1\\0&-1&2\end{bmatrix}$ | 両端を固定した棒の変位。両端が壁に釘止めされたばね列の剛性行列。$-d^2u/dx^2=f$ の $u(0)=0, u(1)=0$ を離散化 |
| $C_n$ | 巡回行列（$K_n$ の四隅に $-1$ を追加） | $\begin{bmatrix}2&-1&-1\\-1&2&-1\\-1&-1&2\end{bmatrix}$ | 輪状に繋がったばね列（周期境界）。温度が環状に分布する系。特異：全体シフトが自由（定数ベクトルが零空間） |
| $T_n$ | 自由-固定（$(1,1)$成分を1に変更） | $\begin{bmatrix}1&-1&0\\-1&2&-1\\0&-1&2\end{bmatrix}$ | 上端が自由、下端が固定のばね。タワー構造（上が自由に動く）。$u'(0)=0, u(1)=0$ の離散化 |
| $B_n$ | 自由-自由（$(1,1)$と$(n,n)$を1に変更） | $\begin{bmatrix}1&-1&0\\-1&2&-1\\0&-1&1\end{bmatrix}$ | 両端とも自由なばね列。全体の平行移動が解に含まれる（特異）。吊り橋の自由端 |

### 差分・分解・変換の記号

| 記号 | 意味 | 具体例 | 何を数値化しているか |
|---|---|---|---|
| $h = \Delta x$ | メッシュ幅 $= 1/(n+1)$ | $n=3$ なら $h=1/4$ | 空間の離散化の細かさ。小さいほど精度↑、計算量↑ |
| $\Delta_+, \Delta_-, \Delta_0$ | 前進・後退・中心差分 | $\Delta_+ u_i = (u_{i+1}-u_i)/h$ | 微分 $du/dx$ の離散近似。前進/後退は1次精度、中心は2次精度 |
| $L, D, U$ | $K = LDL^T$ の下三角・対角・上三角 | $K_3$: $D = \mathrm{diag}(2, 3/2, 4/3)$ | $L$ は消去の手順（乗数）、$D$ はピボット（正定値判定に使う）、$U$ は後退代入用 |
| $Q$ | 直交行列（$Q^T Q = I$） | 回転行列 $\begin{bmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{bmatrix}$ | ベクトルの長さを保つ変換。数値的に安定な分解（QR, SVD）の基盤 |
| $\Sigma$ | 特異値の対角行列 | $A=\begin{bmatrix}1&1\\7&7\end{bmatrix}$ なら $\sigma_1=10, \sigma_2=0$ | 行列が各方向にどれだけ「引き伸ばす」か。$\sigma_{\max}/\sigma_{\min}$ = 条件数 |
| $F_n$ | フーリエ行列 $(F_n)_{jk} = w^{jk}$, $w = e^{2\pi i/n}$ | $F_4 = \begin{bmatrix}1&1&1&1\\1&i&-1&-i\\1&-1&1&-1\\1&-i&-1&i\end{bmatrix}$ | 時間/空間信号を周波数成分に分解。$C_n$ の固有ベクトルが $F_n$ の列（DFT） |

## 重要な数式

### 1.1 四つの特別行列

**$K_n$ の定義**:

$$K_{ij} = \begin{cases} 2 & i=j \\ -1 & |i-j|=1 \\ 0 & \text{otherwise} \end{cases}$$

- 意味: $-d^2/dx^2$ の固定-固定差分近似（$\times h^2$）
- 前提: 境界条件 $u_0 = u_{n+1} = 0$
- 性質: 対称、三重対角、正定値、$\det K_n = n+1$

**$C_n$**: $K_n$ に四隅 $(-1)$ を追加（周期境界）
- 特異（$C_n \mathbf{1} = 0$）、半正定値、零空間 = 定数ベクトル

**$T_n$**: $K_n$ の $(1,1)$ を $1$ に変更
- 意味: 自由端 $u'(0)=0$ ＋ 固定端 $u(1)=0$
- 正定値、$\det T_n = 1$、全ピボット $= 1$

**$B_n$**: $K_n$ の $(1,1)$ と $(n,n)$ を $1$ に変更
- 意味: 両端自由、特異（半正定値）、零空間 = 定数ベクトル

まとめ（PDFより確認）:
> $K_n, T_n$: 可逆、正定値。$C_n, B_n$: 特異、半正定値。零空間は定数ベクトル。

### 1.2 有限差分と精度

**前進差分**（1次精度）:

$$\Delta_+ u = \frac{u(x+h) - u(x)}{h} = u'(x) + \tfrac{1}{2}h\,u''(x) + \cdots$$

**後退差分**（1次精度）:

$$\Delta_- u = \frac{u(x) - u(x-h)}{h} = u'(x) - \tfrac{1}{2}h\,u''(x) + \cdots$$

**中心差分**（2次精度）:

$$\Delta_0 u = \frac{u(x+h) - u(x-h)}{2h} = u'(x) + \tfrac{1}{6}h^2\,u'''(x) + \cdots$$

**2階差分**（2次精度）:

$$\frac{\Delta^2 u}{h^2} = \frac{u_{i+1} - 2u_i + u_{i-1}}{h^2} = u''(x) + \tfrac{1}{12}h^2\,u^{(4)}(x) + \cdots$$

- 役割: $-d^2u/dx^2 = f(x)$ を離散化して $Ku = h^2 f$ に変換する基盤
- 前提: テイラー展開が有効（$u$ が十分滑らか）
- 重要な乗算: $\Delta^2(\text{ones}) = 0$, $\Delta^2(\text{ramp}) = \text{delta}$, $\Delta^2(\text{sines}) = \lambda \cdot \text{sines}$

**境界条件と行列の対応**:

| 境界条件 | 行列 | 離散的設定 |
|---|---|---|
| $u(0)=0,\; u(1)=0$ | $K_n$ | $u_0 = u_{n+1} = 0$ |
| $u'(0)=0,\; u'(1)=0$ | $B_n$ | $u_0 = u_1,\; u_n = u_{n+1}$ |
| $u'(0)=0,\; u(1)=0$ | $T_n$ | $u_0 = u_1,\; u_{n+1} = 0$ |
| 周期 | $C_n$ | $u_0 = u_n,\; u_1 = u_{n+1}$ |

### 1.3 消去法と $K = LDL^T$

**ガウス消去**: $K = LU$

$K_3$ の例（PDFより確認）:

$$\begin{bmatrix} 2 & -1 & 0 \\ -1 & 2 & -1 \\ 0 & -1 & 2 \end{bmatrix} = \begin{bmatrix} 1 & 0 & 0 \\ -\frac{1}{2} & 1 & 0 \\ 0 & -\frac{2}{3} & 1 \end{bmatrix} \begin{bmatrix} 2 & -1 & 0 \\ 0 & \frac{3}{2} & -1 \\ 0 & 0 & \frac{4}{3} \end{bmatrix}$$

**$K_n$ のピボットと乗数**:

$$d_i = \frac{i+1}{i} \quad (i\text{番目のピボット}), \qquad \ell_{i+1,i} = -\frac{i}{i+1}$$

- 意味: 対称行列では $K = LDL^T$ が自然な分解。$D$ の対角 = ピボット
- 役割: ピボットが全て正 $\iff$ 正定値のテスト

**行列式**: $\det K_n = d_1 \cdot d_2 \cdots d_n = \frac{2}{1} \cdot \frac{3}{2} \cdots \frac{n+1}{n} = n+1$

**計算量**:

| | Full | Banded ($w$) | Tridiagonal |
|---|---|---|---|
| Factor | $\frac{2}{3}n^3$ | $2w^2 n + wn$ | $3n$ |
| Solve | $2n^2$ | $4wn + n$ | $5n$ |

### 1.4 逆行列とデルタ関数

**固定-固定 Green関数**:

$$-u'' = \delta(x-a) \implies u(x) = \begin{cases} (1-a)x & x \le a \\ (1-x)a & x \ge a \end{cases}$$

- 意味: 点荷重に対する応答。逆行列の列 = 離散Green関数
- 前提: $u(0) = 0,\; u(1) = 0$

**$K_n^{-1}$ の $(i,j)$ 成分**（$i \le j$ のとき）:

$$(K_n^{-1})_{ij} = \frac{(n+1-j) \cdot i}{n+1}$$

- 対角以下は $i$ と $j$ を入れ替え（$K_n^{-1}$ は対称）

**Woodbury-Sherman-Morrison**:

$$K = T - uv^T \implies K^{-1} = T^{-1} + \frac{T^{-1}uv^T T^{-1}}{1 - v^T T^{-1} u}$$

- 役割: ランク1の修正に対する逆行列の更新

### 1.5 固有値と固有ベクトル

**基本方程式**: $Ax = \lambda x \iff \det(A - \lambda I) = 0$

**$K_n$ の固有値と固有ベクトル**（PDFより確認）:

$$\lambda_k = 2 - 2\cos\frac{k\pi}{n+1}, \quad k = 1, \ldots, n$$

$$y_k = \bigl(\sin k\pi h,\; \sin 2k\pi h,\; \ldots,\; \sin nk\pi h\bigr), \quad h = \frac{1}{n+1}$$

- 意味: 離散サイン（DST）の列が $K_n$ の固有ベクトル
- 前提: 固定-固定境界条件

**$B_n$ の固有値**: $\lambda_k = 2 - 2\cos\frac{k\pi}{n}$,  $k = 0, 1, \ldots, n-1$
- 固有ベクトルは離散コサイン（DCT）

**$C_n$ の固有値**: $\lambda_k = 2 - 2\cos\frac{2\pi k}{n}$,  $k = 0, 1, \ldots, n-1$
- 固有ベクトルは離散フーリエ変換（DFT）の列、$w = e^{2\pi i/n}$

**対角化**:

$$A = S\Lambda S^{-1} \quad \text{（一般）}, \qquad A = Q\Lambda Q^T \quad \text{（対称、} Q \text{は直交）}$$

- 役割: $A^k = S\Lambda^k S^{-1}$、$du/dt = Au$ の解 $u(t) = S e^{\Lambda t} S^{-1} u(0)$

**行列式とトレース**:

$$\det A = \prod_i \lambda_i, \qquad \mathrm{trace}(A) = \sum_i \lambda_i = \sum_i a_{ii}$$

### 1.6 正定値行列

**エネルギーによる定義**: $S$ が正定値 $\iff$ $u^T S u > 0$ for all $u \neq 0$

**二次形式の最小化**:

$$P(u) = \tfrac{1}{2}u^T K u - u^T f$$

$$\text{最小点: } \nabla P = Ku - f = 0 \implies u = K^{-1}f$$

$$P_{\min} = -\tfrac{1}{2}f^T K^{-1} f$$

- 意味: 正定値なら $P(u)$ は bowl shape（唯一の最小点）
- 役割: $Ku = f$ を最小化問題として定式化（Ch8 最適化への橋）

**$A^T C A$ の正定値性**:

$$u^T K u = u^T (A^T C A) u = (Au)^T C(Au) > 0$$

- 前提: $A$ の列が独立（$Au \neq 0$ for $u \neq 0$）かつ $C$ が正定値
- 役割: Ch2 のフレームワーク $K = A^T C A$ が正定値になる条件

**Newton法**:

$$H(u^{i+1} - u^i) = -\frac{\partial P}{\partial u}(u^i)$$

- $H$ = ヘッセ行列（2次微分の行列）、$H$ が正定値なら極小点

### 1.7 数値線形代数: $LU, QR, SVD$

**3つの基本分解**:
1. $A = LU$（消去法 → 下三角 $\times$ 上三角）
2. $A = QR$（直交化 → 直交列 $\times$ 上三角）
3. $A = U\Sigma V^T$（SVD → 直交 $\times$ 対角 $\times$ 直交）

**Gram-Schmidt**: $a_k$ から直交基底 $q_k$ を構成
**Householder**: 反射行列 $H = I - 2uu^T$ で列を消す（数値的に安定）

**SVD**:

$$A = U\Sigma V^T, \quad U^T U = I, \quad V^T V = I, \quad \Sigma = \mathrm{diag}(\sigma_1, \ldots, \sigma_r)$$

- $\sigma_i = \sqrt{\lambda_i(A^T A)}$（特異値 = $A^T A$ の固有値の平方根）
- $Av_i = \sigma_i u_i$, $A^T u_i = \sigma_i v_i$

**ノルムと条件数**:

$$\|A\| = \sigma_{\max}, \quad c(A) = \frac{\sigma_{\max}}{\sigma_{\min}}$$

**$K_n$ の条件数**:

$$c(K_n) = \frac{2 - 2\cos\frac{n\pi}{n+1}}{2 - 2\cos\frac{\pi}{n+1}} \approx \frac{4}{\pi^2}(n+1)^2$$

- 意味: メッシュを細かくすると条件数は $O(n^2)$ で悪化
- 役割: 丸め誤差の増幅率、反復法の収束速度を支配

### 1.8 SVDからの最良基底（PCA/MOR）

**共分散行列**: $\Sigma_n = \frac{1}{n-1}AA^T$ （$A$: データ行列、列 = サンプル）

**PCA**: $AA^T$ の固有ベクトル $u_1, \ldots, u_k$ = 分散最大の $k$ 方向

$$\text{最小残差: } \sum \|a_i - P_k a_i\|^2 = \sigma_{k+1}^2 + \cdots + \sigma_n^2$$

## 論理構造

```
1. 4つの代表行列 Kn, Cn, Tn, Bn の導入
   → 境界条件の違いが構造（可逆性・正定値性・零空間）を決める
   |
2. 有限差分（1階・2階）と精度のオーダー
   → 微分方程式 -u'' = f を離散方程式 Ku = h²f に変換
   → 境界条件が行列の選択 (K/T/B/C) を決める
   |
3. 消去法 → K = LDLᵀ
   → ピボットが正定値性を判定
   → 計算量は構造（full/banded/tridiagonal）で劇的に変わる
   |
4. 逆行列 ↔ Green関数
   → K⁻¹ の列 = 点荷重への応答 = 離散Green関数
   → 連続のGreen関数 u(x,a) と正確に対応
   |
5. 固有値・固有ベクトル
   → Kn の固有ベクトル = 離散サイン → DST
   → Cn の固有ベクトル = 複素指数 → DFT
   → 対角化 A = SΛS⁻¹ → Aᵏ, e^(At) の計算
   |
6. 正定値性
   → エネルギー uᵀKu > 0 が本質的定義
   → 5つの同値テスト（ピボット/固有値/行列式/Cholesky/二次形式）
   → K = AᵀCA の正定値性条件
   → 最小化問題 P(u) = ½uᵀKu - uᵀf の唯一解
   |
7. 数値線形代数: LU → QR → SVD
   → 3つの分解は「構造の利用の仕方」が異なる
   → 条件数 c(K) = σ_max/σ_min が誤差増幅を支配
   → Kn の条件数は O(n²) で成長
   |
8. SVDの応用: PCA, MOR
   → AᵀA の固有ベクトル = 最良基底（分散最大方向）
   → ランクk近似が最良（Eckart-Young）
```

## 解説メモ（ノートブック 260403 より）

### なぜ $K_n$ は「対角2、副対角-1」なのか

$-u'' = f$ をテイラー展開で離散化する：

$$u(x+h) + u(x-h) = 2u + h^2 u'' + O(h^4)$$

奇数次項が消えるため2次精度。移項して $u'' \approx (u_{i+1} - 2u_i + u_{i-1})/h^2$。符号反転で係数 $-1, 2, -1$ → $K_n$ の各行。

- 対角の $2$：「自分」$u_i$ の重み（$-(-2u_i)/h^2$）
- 副対角の $-1$：「左右の隣」$u_{i\pm1}$ の重み

### 固定 vs 自由の境界条件が行列を変える仕組み

$i=1$ の式 $-u_0 + 2u_1 - u_2 = h^2 f_1$ に対して：
- **固定** $u_0 = 0$：$u_0$ が消えて $(1,1)$ 成分は $2$ のまま → $K_n$
- **自由** $u'(0)=0$ → $u_0 = u_1$：代入すると $-u_1 + 2u_1 = u_1$ → $(1,1)$ 成分が $1$ に → $T_n$

### エネルギー $u^T K u$ = ばねの弾性エネルギーの2倍

$K_2$ の例：

$$u^T K u = 2u_1^2 - 2u_1 u_2 + 2u_2^2 = u_1^2 + (u_1 - u_2)^2 + u_2^2$$

右辺 = 壁ばね + 中間ばね + 壁ばね の伸びの二乗和。常に正 → 正定値。

### 平方完成 = 消去法（$LDL^T$）

$K_3$ のエネルギーを平方完成すると：

$$u^T K_3 u = \underbrace{2}_{d_1}\left(u_1 - \frac{1}{2}u_2\right)^2 + \underbrace{\frac{3}{2}}_{d_2}\left(u_2 - \frac{2}{3}u_3\right)^2 + \underbrace{\frac{4}{3}}_{d_3}\,u_3^2$$

各係数 $d_i$ = ピボット。平方完成の手順 = ガウス消去と同一の操作。全 $d_i > 0$ → 正定値。

### $K = LU$ の式変形の仕組み

- 消去の各ステップを行列 $E_1, E_2$ で表す → $E_2 E_1 K = U$
- 逆行列で組み替え：$K = E_1^{-1} E_2^{-1} U = LU$
- $L$ の中身 = 乗数 $\ell_{ij}$ をそのまま詰めるだけ。$E^{-1}$ は符号反転だけ
- $\ell_{31} = 0$ の理由：$K$ が三重対角 → $(3,1)$ がゼロ → 物理的に $u_1$ と $u_3$ の間にばねがない

### $\det K_n = n+1$ の意味

ピボットの積：$\frac{2}{1} \cdot \frac{3}{2} \cdots \frac{n+1}{n} = n+1$（テレスコープ約分）。$\neq 0$ なので可逆。ただし行列式が大きくても安定性は条件数 $c(K_n) \approx (4/\pi^2)(n+1)^2$ で判断。

### 固有値の性質テーブル（p.65）

| 行列の性質 | 固有値 | 固有ベクトル |
|---|---|---|
| 対称 $A^T = A$ | すべて実数 | 直交 |
| 直交 $Q^TQ = I$ | $\|\lambda\| = 1$ | 直交 |
| 正定値 $x^TAx > 0$ | すべて $\lambda > 0$ | 直交 |
| 歪対称 $A^T = -A$ | すべて純虚数 | 直交 |
| マルコフ（行和=1） | $\lambda_{\max} = 1$ | 定常状態 $x > 0$ |
| Toeplitz $-1,2,-1$ | $\lambda_k = 2-2\cos\frac{k\pi}{n+1}$ | $x_k = (\sin\frac{k\pi}{n+1}, \sin\frac{2k\pi}{n+1}, \ldots)$ |

### SVD と固有値の関係

- $\sigma_i^2 = \lambda_i(A^T A)$：特異値の2乗 = $A^TA$ の固有値
- 対称正定値なら $\sigma_i = \lambda_i$
- $V$ の列 = $A^TA$ の固有ベクトル、$U$ の列 = $AA^T$ の固有ベクトル
- 条件数 $c(A) = \sigma_{\max}/\sigma_{\min}$、正定値なら $= \lambda_{\max}/\lambda_{\min}$

### 正定値の5つの同値テスト（まとめ）

1. すべてのピボットが正（$LDL^T$ で確認）
2. すべての首座小行列式が正
3. すべての固有値が正
4. $u^T K u > 0$（$u \neq 0$）
5. $K = A^T A$（$A$ の列が独立）→ Cholesky $A = \mathrm{chol}(K)$ が存在

---

## 不明点 / 飛躍

- **条件数 $c(K_n) \approx \frac{4}{\pi^2}(n+1)^2$ の導出**: $\lambda_{\min} \approx \pi^2 h^2$ の近似が $\cos\theta \approx 1 - \theta^2/2$ に依存しているが、高次項の影響は未検討（PDFでは近似として提示）
- **Householder vs Gram-Schmidt の安定性の差**: 「Householderの方が安定」とあるが、定量的な誤差解析は本章では省略（Section 2.3に委ねている）
- **SVDの計算アルゴリズム**: 実際の計算は `eig(A'*A)` ベースとあるが、LAPACK内部の Golub-Kahan 二対角化への言及は簡潔（詳細は外部文献）
- **PCA/MOR の最適性**: Eckart-Young定理（ランク $k$ 近似の最良性）は述べられているが証明は本章にはない
- **$T_n^{-1}$ の公式**: $K_n^{-1}$ は明示的に与えられるが、$T_n^{-1}$ は $U^T U$ 形式で間接的に構成されており、直接的な閉じた公式は本章では **不確実**（問題として出題されている）
