# Chapter 6 — Initial Value Problems

## 要点
- PDE（偏微分方程式）を「時間発展する初期値問題」として捉える章
- フーリエ表示（周波数分解）と差分法（離散化）の両面から、解の振る舞いと破綻を見通す

## この章の主張
- 初期値問題の本質は空間周波数 \(k\) ごとの時間成長率 — 分散関係 \(\omega=\omega(k)\) が波・減衰・分散を一撃で区別
- PDEの性質（保存・散逸・分散）は周波数成分の扱いで統一理解 — 散逸は高周波を削り、分散はエネルギーを保ったまま形を崩す
- 離散系の増幅率 \(|G(k)|>1\) が計算法の爆発を引き起こす — 安定性はメッシュ比の制約として設計を支配

## 重要キーワード
- 分散関係（dispersion relation）
- 散逸 / 分散 / 保存（dissipative / dispersive / conservative）
- 増幅率 \(G(k)\) ・安定性条件（CFL的制約）
- フーリエ解公式
- Duhamelの原理（ソース項の成長追跡）

## 自分メモ
- 実装前に「保存/散逸/分散のどれが支配的か」を \(\omega(k)\) の形で点検する習慣
- 数値が発散したら「物理由来」vs「離散化の \(|G|>1\)」を切り分け → \(\Delta t\)/\(\Delta x\)/手法選択で対処
- ソース項の扱いをDuhamel的に捉え、誤差の増幅機構を特定する
- 各方程式の \(\omega(k)\) 導出と差分ごとの \(G(k)\) 比較は手元で追い直す必要あり
