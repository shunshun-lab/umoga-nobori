# Introduction — CSEの全体像

## 要点
- 計算科学・工学（CSE）＝現実の問題を数式モデルに落とし、計算で解く営み
- 解析（アイデア）とアルゴリズム（計算）を分離せず一体で学ぶ
- 線形代数・微分方程式・フーリエ・数値計算を「道具箱」として統合

## この章の主張
- 解析と計算は同じ授業で一体として学ぶべき — アイデアはアルゴリズムで明確になり、アルゴリズムはアイデアで選べる
- CSEの基本は **モデル化→離散化→解法** の往復
- 理解は「単純化」で前進する — 非線形→線形、連続→離散、多次元→一次元、変係数→定係数

## 重要キーワード
- 線形化（linearization）
- 離散化（finite difference / finite element）
- 安定性（stability）
- 疎行列 K・解法（LU / 反復 / 前処理）
- フーリエ変換・FFT

## 他章とのつながり
- Ch1–2：線形代数・フレームワーク \(K=A^TCA\) として具体化
- Ch3：境界値問題＝離散化の実践
- Ch4–5：フーリエ・解析関数＝道具箱の拡張
- Ch6：安定性の中心舞台（初期値問題）
- Ch7–8：解法・最適化＝「どう解くか」の実装面

## 自分メモ
- 問題を「モデル化／離散化／解法」に切って詰まり箇所を特定する習慣
- 精度と安定は別物 — 当てはまりが良いのに発散するのは安定性の問題
- 近似の説明責任：線形化や定係数化を「便利な嘘」として位置づけ、適用範囲を言語化する
- 「単純化をやめて複雑さへ戻す」判断基準は後続章に委ねられている
