# アルゴリズム選択マップ

> **原則**: 力任せに解くより、構造に合った解法を選ぶ方が投資対効果が高い（Intro）
>
> このファイルは本書で登場するアルゴリズムを「問題の構造 → 解法」の判断順で整理したもの。

## 0. 判断フロー（最初に見る）

```
問題を受け取る
  │
  ├─ 何を解くか？
  │    ├─ Ku=f（静的平衡）      → §1 へ
  │    ├─ 時間発展 PDE          → §2 へ
  │    ├─ 関数の近似・補間       → §3 へ
  │    └─ 最適化（最小化＋制約） → §4 へ
  │
  └─ K の構造を確認（§1 の前に必ずやる）
       ├─ 疎か密か？
       ├─ 帯行列か？ 帯幅は？
       ├─ 対称か？ 正定値か？
       ├─ 定数対角（Toeplitz/circulant）か？
       └─ 規模は？（N, 次元）
```

## 1. 線形方程式 Ku=f の解法

### 1a. 直接法


| 解法               | 前提条件       | 計算量（目安）                  | メモリ     | 使いどころ                | 出典       |
| ---------------- | ---------- | ------------------------ | ------- | -------------------- | -------- |
| **密 LU 分解**      | 制約なし（密行列）  | O(n³)                    | O(n²)   | 小規模、構造なし             | Ch1, Ch7 |
| **帯行列 LU**       | 帯幅 w の帯行列  | O(nw²)                   | O(nw)   | 1D差分（w小）、帯が狭い場合      | Ch7      |
| **コレスキー分解**      | 対称正定値（SPD） | O(n³)/6 （密）、帯なら O(nw²)/2 | 密の半分    | SPD なら LU より常に優先     | Ch1      |
| **リオーダリング + LU** | 疎行列（2D/3D） | フィルイン依存                  | フィルイン依存 | 並べ替えでフィルインを減らしてから LU | Ch7      |


**判断ポイント**:

- 1D・帯幅小 → 帯行列 LU で十分（O(n) に近い）
- 2D自然順序 → 帯幅 ≈ N、計算量 ≈ N⁴ → 並べ替え必須 or 反復法へ
- 3D → 直接法はほぼ破綻 → 反復法一択

### 1b. 反復法


| 解法                 | 前提条件       | 1反復のコスト       | 収束               | 使いどころ            | 出典  |
| ------------------ | ---------- | ------------- | ---------------- | ---------------- | --- |
| **ヤコビ法**           | 対角優位なら収束保証 | O(nnz)        | 遅い               | 教育・並列の原型         | Ch7 |
| **ガウス・ザイデル / SOR** | SPD で収束保証  | O(nnz)        | ヤコビより速い（SOR は加速） | 中規模・構造的に良い場合     | Ch7 |
| **共役勾配法（CG）**      | SPD 必須     | O(nnz)        | 条件数依存 √κ         | **SPD 大規模の標準解法** | Ch7 |
| **GMRES**          | 非対称でも可     | O(nnz × iter) | リスタート要           | 非対称・不定の大規模系      | Ch7 |
| **マルチグリッド**        | 楕円型 PDE 由来 | O(n)          | 最速（メッシュ非依存）      | **大規模 PDE の本命**  | Ch7 |


**判断ポイント**:

- SPD → まず CG + 前処理
- 非対称 → GMRES（またはBiCGSTAB）
- 楕円型 PDE → マルチグリッドが理想（O(n) が可能）
- **前処理が反復法の成否を握る**（前処理なしの CG は遅い）

### 1c. 変換法


| 解法                | 前提条件                   | 計算量        | 使いどころ            | 出典            |
| ----------------- | ---------------------- | ---------- | ---------------- | ------------- |
| **FFT（高速フーリエ変換）** | 周期境界 + 定数係数（circulant） | O(n log n) | 周期問題・畳み込み・スペクトル法 | Ch1, Ch4, Ch6 |


**判断ポイント**:

- 定数対角（Toeplitz/circulant）→ FFT が最速
- 周期境界でなくても、DST/DCT で対応できる場合がある

## 2. 時間発展 PDE の離散化

### 2a. 時間積分スキーム


| スキーム                       | 型   | 安定性           | 精度  | トレードオフ         | 出典  |
| -------------------------- | --- | ------------- | --- | -------------- | --- |
| **前進オイラー（Forward Euler）**  | 陽的  | 条件付き（Δt 制約あり） | 1次  | 簡単だが Δt が小さくなる | Ch6 |
| **後退オイラー（Backward Euler）** | 陰的  | 無条件安定         | 1次  | 毎ステップ連立方程式を解く  | Ch6 |
| **クランク・ニコルソン**             | 陰的  | 無条件安定         | 2次  | 精度と安定の標準的バランス  | Ch6 |
| **ルンゲ・クッタ（RK4 等）**         | 陽的  | 条件付き          | 4次  | 高精度だが Δt 制約は残る | Ch6 |
| **リープフロッグ**                | 陽的  | 条件付き（分散型向き）   | 2次  | 波動方程式に適合       | Ch6 |


### 2b. 安定性の判断


| チェック項目         | 基準                                   | 違反すると                        | 出典  |
| -------------- | ------------------------------------ | ---------------------------- | --- |
| **増幅率 |G(k)|** | すべての k で |G| ≤ 1                     | 数値解が発散（物理ではなく計算が爆発）          | Ch6 |
| **CFL 条件的制約**  | Δt/Δx² ≤ R_max（拡散）、Δt/Δx ≤ C_max（移流） | 高周波から不安定化                    | Ch6 |
| **散逸 vs 分散**   | 散逸型 → 高周波を殺す、分散型 → エネルギー保存で形が崩れる     | スキームと PDE の性質の不一致 → 偽の振動や過減衰 | Ch6 |


**判断ポイント**:

- 拡散型 PDE → 陰的（Δt 制約を外せる）が実用的
- 波動型 PDE → 陽的（リープフロッグ等）で分散特性を保存
- 陽的で Δt 制約がきつすぎる → 陰的へ切り替え（§1 の連立方程式解法が必要になる）

## 3. 関数の近似・補間


| 手法                     | 前提           | 精度           | 落とし穴                     | 出典       |
| ---------------------- | ------------ | ------------ | ------------------------ | -------- |
| **フーリエ級数**             | 周期関数         | 滑らかなら指数的     | 不連続 → Gibbs 現象（係数減衰 1/k） | Ch4      |
| **FFT（離散フーリエ変換）**      | 周期 + 等間隔サンプル | O(n log n)   | エイリアシング                  | Ch4      |
| **多項式補間（等間隔点）**        | なし           | 次数依存         | **Runge 現象**（端で暴れる）      | Ch5      |
| **チェビシェフ点 + バリセントリック** | 解析的な関数       | 指数的          | 特異点が近いと鈍化                | Ch5      |
| **スペクトル法**             | 解析的 + 適切な境界  | 指数的          | 実装が複雑、非滑らかに弱い            | Ch5      |
| **有限要素法（FEM）**         | 弱形式に落とせる     | 多項式次数依存（代数的） | 要素設計・メッシュ依存              | Ch3      |
| **有限差分法**              | 格子上で定義       | Δx の冪（1次/2次） | 不規則形状に弱い                 | Ch1, Ch3 |


**判断ポイント**:

- 解が滑らか + 周期的 → フーリエ / スペクトル法（指数精度）
- 解が滑らか + 非周期 → チェビシェフ点（指数精度）
- 解に不連続がある → FEM / 有限差分（代数精度で我慢）+ Gibbs 対策
- **精度の限界は複素平面での特異点の位置で決まる**（Ch5）

## 4. 最適化


| 手法                     | 問題の型             | 核心                     | 出典  |
| ---------------------- | ---------------- | ---------------------- | --- |
| **法線方程式 A^TAx = A^Tb** | 線形最小二乗           | 射影 + 直交条件（残差 ⊥ 列空間）    | Ch8 |
| **正則化（Tikhonov 等）**    | 不適切問題（ill-posed） | ペナルティ項で「解ける・壊れない」形に変える | Ch8 |
| **ラグランジュ乗数 / KKT**     | 等式・不等式制約付き最適化    | 制約 → 鞍点行列（KKT行列）を解く    | Ch8 |
| **随伴法（adjoint）**       | 設計変数が多い勾配計算      | 勾配を変数の数に依存せず一度で計算      | Ch8 |
| **Euler–Lagrange**     | 汎関数の最小化（連続体）     | 弱形式の最適性条件 → PDE に帰着    | Ch8 |


**判断ポイント**:

- 最小二乗で解が暴れる → 正則化を足す
- 制約がある → KKT / ラグランジュ乗数（鞍点問題になる）
- 設計変数が多い → 有限差分で勾配を出すと O(変数数) 回解く必要 → **随伴法で 1 回に**
- 連続体の最適化 → Euler–Lagrange → 結局 PDE を解く問題に戻る

## 5. 構造別クイックリファレンス

> 「この構造が見えたら、まずこれを試せ」


| 構造              | 第一選択         | 避けるべきこと            | 規模の壁       |
| --------------- | ------------ | ------------------ | ---------- |
| 密・小規模（n < 1000） | LU / コレスキー   | —                  | なし         |
| 帯行列・1D          | 帯 LU         | 密 LU（無駄）           | なし         |
| SPD・大規模         | CG + 前処理     | 密 LU（メモリ死）         | 前処理の質      |
| 非対称・大規模         | GMRES + 前処理  | CG（SPD でないと壊れる）    | リスタート戦略    |
| 周期 + 定数係数       | FFT          | 直接法（O(n³) は過剰）     | なし         |
| 2D/3D 楕円型 PDE   | マルチグリッド      | 自然順序 LU（N⁴ 級）      | 実装の複雑さ     |
| 不適切逆問題          | 正則化 + CG/直接法 | 正則化なしの最小二乗（暴れる）    | 正則化パラメータ選択 |
| 設計最適化（多変数）      | 随伴法 + 反復     | 有限差分勾配（O(変数数) 回解く） | 随伴の導出コスト   |

## 6. コスト早見表


| 手法         | 計算量        | メモリ     | 備考                    |
| ---------- | ---------- | ------- | --------------------- |
| 密 LU       | O(n³)      | O(n²)   |                       |
| 帯 LU（帯幅 w） | O(nw²)     | O(nw)   | 1D 差分なら w=O(1) → O(n) |

| 随伴法（勾配1回）  | ≈ 順問題2回分   | 順問題と同程度 | 変数数に非依存               |
