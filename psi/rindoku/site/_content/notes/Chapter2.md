# Chapter 2 — A Framework for Applied Mathematics

## 要点
- 応用問題を共通骨格 \(e=Au\), \(w=Ce\), \(f=A^Tw\) → \(K=A^TCA\) で組み立て直す章
- 分野が違っても「局所ルールの積み上げ＝全体方程式」に統一できる

## この章の主張
- 多くの問題は接続 \(A\) と素材則 \(C\) を見つければ \(K=A^TCA\) に統合できる
- 難しさは解法より先に構造で決まる — 平衡なら \(Ku=f\)、時間依存なら \(Mu''+Ku=f(t)\) へ拡張しても骨格は不変
- 同じ \(K\) がばね列（剛性）、最小二乗、グラフ/回路、構造物の平衡、共分散まで翻訳する

## 重要キーワード
- \(e=Au\) / \(w=Ce\) / \(f=A^Tw\)（状態→変化→法則→外力）
- 剛性行列 \(K=A^TCA\)
- 最小二乗（長方行列）
- グラフモデル・キルヒホッフ則
- 非線形 / 時間依存（動力学）

## 他章とのつながり

- Ch7：\(K\) の大規模化と解法選択

## 自分メモ
- 新しい現象でも「状態→接続→材料法則→釣り合い」に分解 → ボトルネック特定
- 実装は \(A\) と \(C\) を作って \(K=A^TCA\) を組む手順に揃える → テストも局所/全体で二分
- 初見の系で「どこを状態 \(u\) と呼ぶか」は慣れが要る
