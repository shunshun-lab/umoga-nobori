## 線形代数コア（全章で効く）

| 日本語 | English | メモ |
|--------|---------|------|
| 可逆（行列・写像） | invertible | 正方行列なら **\(A^{-1}\) が存在**する命题とほぼ同義。写像としては「全単射」。計算上は「一意に解ける」こととセットで覚えると Ch1 の \(Ku=f\) の文脈に直結する。 |
| 非可逆 | non-invertible | **解が一意にならない／存在しない**可能性が出る入口語。実装では「ピボットが出ない」「小さな数値ピボット」に化けやすい（数学の特異とIEEEのゼロ割は別だが、症状は似る）。 |
| 非特異 | nonsingular | 英語圏の線形代数では **invertible とほぼ同語**。論文では nonsingular の方が形容詞として出やすい。**「非特異＝良い」**の対義は singular。 |
| 特異（行列） | singular | **不可逆**の行列。**零空間が自明でない**（非自明な \(Ax=0\)）と同値、など「同値語の束」で理解すると Appendix と一本化できる。 |
| 正則行列 | nonsingular matrix; invertible matrix | 日本語教材の「正則」は、この2語のどちらにでも訳されがち。**正定値 positive definite**（二次形式が正定）とは別物——記号が似ているので要注意。 |
| 特異行列 | singular matrix | \( \det A=0 \)（正方の場合）／ランク低下など。**物理ではしばしば「ゲージ自由度」「剛体モード」**として現れる（Ch1 の \(C_n,B_n\) の定数モード等）。 |
| フルランク | full rank | 行列の形 \((m\times n)\) に対し **rank = min(m,n)**。長方行列では「列が満秩」版和文で言いがち。**可逆は正方フルランクに限定**される。 |
| ランク不足・ランク欠損 | rank-deficient | full rank でないこと。**データサイドは情報不足**、モデル側は**余分なパラメータ（欠損ランク）**の両方で起きる。最小二乗なら法方程式が特異になりやすい。 |
| 条件数が悪い | ill-conditioned | **行列は可逆でも「逆に触れると壊れる」**状態。誤差が \(\kappa(A)\) 倍され得るという意味。**特異に近い**とは区別して語る慣習。 |
| 良性・適切な条件 | well-conditioned | \(\kappa\) が小さめで、入力摂動に対して解が暴れにくい。**スケーリング（行・列の単位）**で劇的に改善することが多い実務テーマ。 |
| 零空間・核 | null space; kernel | \(Ax=0\) の解空間。**物理では未拘束モード／バランス欠落**のシグナル。FEM・バネネットワークで「なぜ解が定まらないか」の説明に効く。 |
| 列空間・値域 | column space; range | \(b\) が \(Ax=b\) で**到達可能か**を決める空間。**\(b \notin C(A)\)** なら最小二乗へ、が Ch2–8 の流れ。 |
| ランク | rank | 独立な列（または行）の本数。「モデルの実効自由度」の感覚。**数値ランク**（閾値で切る）と混ぜない。 |
| 行列式 | determinant | 正方で「体積の拡大率」。**大きさ自体より「ゼロかどうか」**が可逆判定で効くことが多い。大規模計算では det よりランク・固有値。 |
| ピボット | pivot | 消去の各段で割る主成分。**小さいピボット = 不安定／条件が悪い**。正定値では \(LDL^T\) の対角と一体化して理解できる（Ch1）。 |
| 正定値 | positive definite |（実対称が主）**エネルギー二次形式 \(u^TSu>0\)**。離散では「剛性がちゃんと効いている＝安定な弾性系」イメージ。**可逆性（一般）より強い**要求。 |
| 半正定値 | positive semidefinite | **\(u^TSu\ge 0\)**。零空間を許す＝**剛体モードあり**の系で自然。Cholesky は特異で止まるので半正定では一般に要工夫。 |
| 直交行列 | orthogonal matrix | \(Q^TQ=I\)。**長さと角度を保つ**ので数値が崩れにくい基底変換。QR・SVD の「骨格」。複素版はユニタリ。 |
| 特異値 | singular value | SVD の対角 \(\sigma_i\)。**行列が各方向にどれだけ伸ばすか**のスカラー。**singular matrix とは無関係**（紛らわしいが別ルートの形容詞）。 |
| 特異値分解 | singular value decomposition; **SVD** | \(A=U\Sigma V^T\)。**秩・疑似逆・低ランク近似（カットオフ）**の共通ツール。最小二乗の「構造的な答え」を与える。 |
| 擬似逆 | Moore–Penrose inverse; **pseudoinverse** | \(A^+\)。**最小ノルム最小二乗解**など、ランク欠損でも一意に「一番まともな」逆を選ぶ道具。正則化（Ch8）と発想がつながる。 |

## Introduction — CSEの全体像

| 日本語 | English | メモ |
|--------|---------|------|
| 計算科学・工学 | computational science and engineering; **CSE** | 数学モデル・数値法・実装を一体で扱う分野ラベル。本の趣旨は「式の理解とアルゴリズム選択を往復する」こと。 |
| モデル化 | modeling | 現象を **未知数・パラメータ・方程式**に写す工程。ここでの仮定が後の離散化の“許容誤差”を決める。 |
| 離散化 | discretization | 連続問題を有限自由度へ。**安定性・境界・保存則**がここで盛り込まれ、後から直せないことが多い。 |
| 有限差分法 | finite difference method; **FDM** | 格子点上で微分を差分に置換。**実装は軽いが境界が複雑だと詰まりやすい**。Ch1→3 の足場。 |
| 有限要素法 | finite element method; **FEM** | 試験関数とメッシュで弱形式を離散化。**複雑形状・境界条件に強い**代わりに前処理・行列組み立てが重い。 |
| 線形化 | linearization | 非線形を **局所的に一次式**へ。**Newton 反復の1ステップ**とも重なる。適用範囲外だと解の分枝を取りこぼす。 |
| 安定性（離散化・時間積分） | stability | **数値が暴れないこと**（例：\(\|G\|>1\)）。誤差が小さいこと（精度）とは独立に議論される——Intro の重要メッセージ。 |
| 精度・誤差 | accuracy; error | **大域誤差／離散化誤差／丸め誤差**など層がある。安定かつ精度設計が揃って初めて「信じられる答え」。 |
| 解法 | solver; solution method | 直接法・反復法・固有値ソルバ等の総称。**行列の構造（疎・対称・SPD）**で選ぶのが Ch7 の主戦場。 |
| 疎行列 | sparse matrix | 非ゼロが少ない表現。**アルゴリズムは密行列の常識が通用しない**（順序・フィルインが命）。 |
| フーリエ変換 | Fourier transform | 周波数表示への基底変換。**PDE の分散関係・スペクトル法**の共通言語（Ch4–6）。 |
| 高速フーリエ変換 | Fast Fourier Transform; **FFT** | DFT を \(O(n\log n)\) で。**巡回構造・畳み込み・スペクトル系列**で実装の主役になりやすい。 |

## Chapter 1 — Applied Linear Algebra

| 日本語 | English | メモ |
|--------|---------|------|
| 作用素としての行列 | matrix as a linear operator | Ch1 の態度は「数字の表」より **\(u \mapsto Ku\)**。**境界条件込みの離散化**が \(K\) の形を決める。 |
| 固定-固定剛性行列 \(K_n\) | fixed–fixed stiffness matrix \(K_n\) | 2階差分＋両端ディリクレの定番。**SPD（対称正定）・可逆**。\(\det K_n=n+1\) など具体量があるとデバッグに効く。 |
| 巡回剛性行列 \(C_n\) | periodic / circulant stiffness \(C_n\) | **周期境界**。**半正定・特異**、零空間は定数ベクトル（剛体シフト）。固有ベクトルは **DFT** 列（\(F_n\)）。 |
| 自由-固定 \(T_n\) | free–fixed (\(T_n\)) | 片端ノイマン片端ディレの典型。**正定・可逆**（本書の対応表の通り）。 |
| 自由-自由 \(B_n\) | free–free (\(B_n\)) | 両端ノイマン的。**半正定・特異**、再び定数モード。**「外れ値を抑える」**等でピン留めしないと壊れがち。 |
| 剛性行列（一般名） | stiffness matrix | 離散／FE での **全体剛性** をまとめた \(K\)。\(K_n,C_n,T_n,B_n\) は**境界条件別の代表形**（英語では文脈で fixed–fixed / free–free / periodic と説明すると通じやすい）。 |
| メッシュ幅 | mesh spacing; grid spacing | \(h=\Delta x\)。**誤差は \(h\) の冪**、条件数やスケールは \(h\) 依存——単位を揃えないと条件数だけ見て誤診する。 |
| 2階中央差分（離散ラプラシアン） | second central difference; discrete Laplacian | \((u_{i+1}-2u_i+u_{i-1})/h^2\)。\(-d^2/dx^2\) の2次精度近似。**ポアソン型 \(Ku=h^2f\)** の心臓部。 |
| ガウス消去 | Gaussian elimination | \(K=LU\) の手続き。**ピボットが小さい／ゼロ**は数学的特異／数値特異のサイン。 |
| コレスキー分解 | Cholesky factorization | SPD なら **\(K=LL^T\)**（または \(R^TR\)）。\(LDL^T\) の「正定が見えた版」。半正定だと対角に0が出て止まることがある。 |
| 三重対角 | tridiagonal | 帯幅1。**分解・求解は \(O(n)\)** 台（Ch1 の計算量表の世界）。多次元で帯幅が増えると Ch7 へ。 |
| 帯行列 | band matrix | 非ゼロが対角帯に乗る構造。**帯幅 \(w\)** が因子分解コストの目安になる。 |
| ピボット（消去） | pivot | \(LDL^T\)／LU で割る対角成分。**全て正**なら（対称の典型的流儀で）正定の判定に直結。 |
| スケーリング（行・列） | scaling; equilibration | 物理単位の取り違えで **見かけの条件数**が悪化——Ch1 の「条件数＝本質？」の実務的補足。 |
| 離散サイン変換（固有ベクトル） | discrete sine transform; **DST** | \(K_n\) の固有ベクトルがサイン波サンプル。**FFT 族**で速く回せる＝直接法以外の道筋。 |
| 離散コサイン変換 | discrete cosine transform; **DCT** | \(B_n\) 周りで自然。**境界が片側「偶延長」**気味の問題と相性。 |
| 離散フーリエ変換 | discrete Fourier transform; **DFT** | \(C_n\) の固有空間。**周期境界＝周波数がそのままモード**。Ch4・Ch6 への橋。 |
| フーリエ行列 | Fourier matrix | \(F_n\)、\(w=e^{2\pi i/n}\)。**単位根**の直交（正規化込みでユニタリ）。 |
| 対角化 | diagonalization | \(A=S\Lambda S^{-1}\)。対称なら **\(Q\Lambda Q^T\)**（実直交）。**\(A^k\)、\(\exp(tA)\)** の閉形式に効く。 |
| トレース | trace | \(\sum_i \lambda_i = \sum_i a_{ii}\)。**不変量**として検算に使える。 |
| エネルギー二次形式 | energy quadratic form | \(\frac12 u^T Ku - u^T f\)。**弾性エネルギー − 外力の仕事**。正定 \(K\) なら唯一解＝最小エネルギー原理と同値（弱形式の片鱗）。 |
| Sherman–Morrison–Woodbury | Sherman–Morrison–Woodbury formula | **ランク1（低ランク）修正**の逆行列更新。境界を1箇所いじった、行列を1列いじった、の差分に効く。 |
| 前進差分 | forward difference | **1次精度**の \(u'\) 近似。移流では **upwind** として非対称化されることも（別章）。 |
| 後退差分 | backward difference | 1次精度。**暗黙スキーム**や片側情報に寄せた近似で対に出る。 |
| 中心差分（1階） | centered (central) difference | **2次精度**の \(u'\) 近似。**2階中央差分（ラプラシアン）**とは別レシピ——混同しない。 |
| \(LDL^T\) 分解 | \(LDL^T\) factorization | 対称行列向け。**ピボット＝正定判定**に直結。スパースでは順序とセット（Ch7）。 |
| \(LU\) 分解 | \(LU\) factorization | 一般正方。**ピボット選出**で安定性が変わる。密小規模〜中規模の基準解法。 |
| \(QR\) 分解 | \(QR\) factorization | **直交化**。最小二乗・固有値（反復）の土台。条件数改善のイメージを持つとよい。 |
| 条件数 | condition number | SPD では **\(\lambda_{\max}/\lambda_{\min}\)** が代表的。**細かいメッシュほど最小固有値が小さく**なりやすく、見かけが悪化しやすい（スケーリング注意）。 |
| Green関数 | Green's function | 連続：\(-u''=\delta\) 型の基本解。離散：**\(K^{-1}\)** の列／要素が「点荷重の応答」。感度・影響半径の直感に使う。 |
| 固有値・固有ベクトル | eigenvalue; eigenvector | \(Ax=\lambda x\)。**離散モード形状**（サイン／コサイン／複素指数）。時間発展・振動数・条件数の三者をつなぐホブ。 |

| 日本語 | English | メモ |
|--------|---------|------|
| ひずみ・変形（離散） | strain; elongation | \(e=Au\)：**局所差・勾配**の離散像。物理場では「どれだけ伸びたか」。 |

## Chapter 3 — Boundary Value Problems

| 日本語 | English | メモ |
|--------|---------|------|
| 境界値問題 | boundary value problem; **BVP** | 空間全体で一度に解く。**端での情報が全局を規定**するのが差分・FEMの世界観。 |
| 初期値問題（対比用） | initial value problem; **IVP** | 時間方向に積む典型。**安定性の議論の舞台が時間ステップ**へ移る（Ch6）。 |
| 弱形式 | weak form; variational form | 微分を試験関数側に移す。**低正則性の解を許す**＝FEM の出発点。 |
| 試験関数 | test function | 「任意の \(v\) で…」の \(v\)。**メッシュ上不適切な微分**を避けるための工夫の核。 |
| 勾配（1次元） | derivative; gradient | \(u'\)。高次元では \(\nabla u\)。**エネルギーは勾配で評価**されがち。 |
| フラックス | flux | \(cu'\) 型の**流れ／流束**。連続条件（境界での継ぎ目）が実装バグの温床。 |
| ディリクレ境界条件 | Dirichlet boundary condition | **値を直接指定**（例：\(u=g\)）。行列では「行を潰して1を置く」等の同値操作と対応。 |
| ノイマン境界条件 | Neumann boundary condition | **勾配／フラックス指定**。自然境界条件として弱形式に「勝手に入る」場合も。 |

## Chapter 4 — Fourier Series / Integrals

| 日本語 | English | メモ |
|--------|---------|------|
| フーリエ級数 | Fourier series | 周期関数を**三角／指数基底**に分解。**直交性＝係数は投影**（最小二乗と同型）。 |
| フーリエ係数 | Fourier coefficients | 「各モードの振幅」。**滑らかさが減衰率**に効く（高周波が残る＝尖り）。 |
| 直交性 | orthogonality | 異なる基底の内積が0。**基底展開の計算を「積分一発」にする装置**。 |
| 直交基底 | orthogonal basis | 解析と数値の両方で扱いやすい。**正規直交化で条件がさらに良くなる**。 |
| パーセバルの定理 | Parseval's theorem | **時空間エネルギー＝周波数エネルギー**の恒等式。物理的解釈に強い。 |
| Gibbs現象 | Gibbs phenomenon | 不連続付近の**過大振動**。項数を増やしても「ジャンプを丸く誤魔化す」限界がある、が理解の鍵。 |
| 畳み込み | convolution | 積→和の線形操作。**フィルタ・拡散（熱核）・確率の和**まで同じ形。FFT とセット。 |

## Chapter 5 — Analytic Functions

| 日本語 | English | メモ |
|--------|---------|------|
| 解析関数 | analytic function | 級数（テイラー）で**局所から大域へ解析接続**できる関数クラス。実関数の「無限回微分可能」とは別物の罠あり。 |
| 正則関数・全純 | holomorphic function | 複素微分可能（コーシー・リーマン）。**本書では holomorphic ≒ analytic** として読んでよい。 |
| テイラー級数 | Taylor series | **局所データ（全微係数）が遠方を規定**する装置。**収束半径＝最近傍の特異点**まで。 |
| ローラン級数 | Laurent series | 負の冪を許す＝**極や本性特異点**を記述。留数は係数の抜き出し。 |
| 収束半径 | radius of convergence | 級数が収束する最大円。**特異点までの距離**がそのまま数値の「届く範囲」。 |
| 特異点 | singularity | **複素関数が壊れる点**（極・分岐・本性など）。**行列の singular（不可逆）とは別概念**——綴りが同じなので要注意。 |
| 極 | pole | \(\lvert f\rvert\to\infty\) の典型的特異。**留数定理**で積分が代数化する。 |
| 留数 | residue | 極まわりの1次の係数。**閉曲線積分＝\(2\pi i\)×（留数和）**の計算芯。 |
| コーシーの積分定理 | Cauchy's theorem | 解析なら**閉路積分ゼロ**。経路変形の自由が「技巧の源泉」。 |
| コーシーの積分公式 | Cauchy's integral formula | 境界の値から内部の値・**導関数まで**を積分で回収＝ディスク上の再構成原理。 |
| スペクトル法 | spectral method | グローバルな滑らかな基底（三角・多項式）で**指数収束**を狙う。**解析性が前提**、不連続だと苦戦。 |
| チェビシェフ点 | Chebyshev points | 区間上の補間ノード。**端に密**＝Runge 緩和の定番。 |
| バリセントリック補間 | barycentric interpolation | 補間多項式を**安定な重み付き比**で評価。**高次でも係数を明示しない**のが実務で有利。 |
| 指数精度 | exponential accuracy; spectral accuracy | 誤差が **\(O(r^{-N})\)（\(r>1\)）**系。**特異点が遠いほど \(r\) が大きく**取れるイメージ。 |

## Chapter 6 — Initial Value Problems

| 日本語 | English | メモ |
|--------|---------|------|
| 分散関係 | dispersion relation | \(\omega(k)\)：各モードの**時間振る舞い**（振動・減衰）を一括指定。 |
| 散逸型 | dissipative | **振幅が減る**機構（粘性など）。高周波が落ちる＝「なめらか化」も起きうる。 |
| 分散型 | dispersive | **位相速度が \(k\) 依存**→波形が崩れるがエネルギーは巡りがち。 |
| 保存系 | conservative | 散逸なし（ハミルトン的）。**シンプレクティック等の保存構造**が選ばれる理由。 |
| 増幅因子 | amplification factor | 1ステップの **\(G(k)\)**。**\(\lvert G\rvert\le 1\)** が安定の骨格（厳密には法・方程式依存）。 |
| CFL条件 | Courant–Friedrichs–Lewy condition; **CFL condition** | 陽的風のスキームで現れる **\(\Delta t \lesssim C \Delta x\)** 型。**情報が1ステップで届かないと発散**の直感。 |
| 離散安定性 | numerical stability | 丸め誤差・離散化が**指数増幅しない**こと。解析安定≠実装安定（境界・実装バグ注意）。 |
| Duhamelの原理 | Duhamel's principle | 非同次（ソース）を**インパルス応答の重ね合わせ**に帰着。Green 的発想の時間版。 |

## Chapter 8 — Optimization

| 日本語 | English | メモ |
