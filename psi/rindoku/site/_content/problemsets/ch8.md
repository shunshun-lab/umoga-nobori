## PS 8.1 — 最小二乗と法線方程式

**問題**: A = [[1,1],[1,2],[1,3]], b = (1, 2, 2)ᵀ のとき、Ax ≈ b の最小二乗解を求め、残差が列空間に直交することを確認せよ。

**回答**:

$$A^TA = \begin{bmatrix}3 & 6\\6 & 14\end{bmatrix}, \quad A^Tb = \begin{bmatrix}5\\11\end{bmatrix}$$

$$\begin{bmatrix}3&6\\6&14\end{bmatrix}\begin{bmatrix}\hat{x}_1\\\hat{x}_2\end{bmatrix} = \begin{bmatrix}5\\11\end{bmatrix}$$

det = 42 − 36 = 6。x̂₁ = (5·14 − 6·11)/6 = (70−66)/6 = 2/3、x̂₂ = (3·11 − 6·5)/6 = 3/6 = 1/2。

最小二乗解: **x̂ = (2/3, 1/2)**。近似直線: y = 2/3 + x/2。

残差: r = b − Ax̂ = (1, 2, 2) − (7/6, 10/6, 13/6) = (−1/6, 2/6, −1/6)。

直交確認: Aᵀr = (−1/6 + 2/6 − 1/6, −1/6 + 4/6 − 3/6) = (0, 0) ✓。

**核心**: 最小二乗の最適性条件 Aᵀr = 0 は「残差 ⊥ 列空間」。これが射影の幾何学的意味。

## PS 8.2 — Tikhonov 正則化

**問題**: Ax = b が不適切（ill-posed）なとき、正則化 min ‖Ax−b‖² + α‖x‖² がどう安定化するか説明せよ。

**回答**:

正則化された法線方程式:

$$(A^TA + \alpha I)\hat{x}_\alpha = A^Tb$$

**SVD で理解**: A = UΣVᵀ のとき、

$$\hat{x}_\alpha = \sum_i \frac{\sigma_i}{\sigma_i^2 + \alpha}(u_i^Tb)\,v_i$$

正則化なし（α=0）: x̂ᵢ = (uᵢᵀb)/σᵢ。σᵢ が小さいとノイズが 1/σᵢ 倍に増幅 → 解が暴れる。

正則化あり: σᵢ/(σᵢ² + α) は σᵢ >> √α のとき ≈ 1/σᵢ（そのまま）、σᵢ << √α のとき ≈ σᵢ/α → 0（抑制）。

**α の選び方**:
- L曲線法: ‖x̂_α‖ vs ‖Ax̂_α − b‖ のプロットの「肘」
- GCV（一般化交差検証）
- Morozov の不一致原理

**核心**: 正則化は「解ける形に変える」道具。不適切 → 適切への変換。Ch4 のデコンボリューションで見た問題の一般的解決策。

## PS 8.3 — Euler-Lagrange 方程式

**問題**: 汎関数 J[u] = ∫₀¹ (½(u')² − fu) dx を最小にする u(x) が −u'' = f を満たすことを示せ（u(0)=u(1)=0）。

**回答**:

変分: u → u + εv（v(0)=v(1)=0）を代入:

$$J[u+\varepsilon v] = \int_0^1 \left(\frac{1}{2}(u'+\varepsilon v')^2 - f(u+\varepsilon v)\right)dx$$

ε で微分して ε=0 とおく:

$$\frac{d}{d\varepsilon}J[u+\varepsilon v]\Big|_{\varepsilon=0} = \int_0^1 (u'v' - fv)\,dx = 0$$

部分積分: ∫ u'v' dx = −∫ u''v dx + [u'v]₀¹ = −∫ u''v dx（v(0)=v(1)=0）。

$$\int_0^1 (-u'' - f)v\,dx = 0 \quad \text{すべての } v \text{ に対して}$$

基本補題: **−u'' = f**。

**核心**: 最小化 → Euler-Lagrange 方程式 → PDE。「解を求める」と「エネルギーを最小にする」は同値。Ch3 の弱形式の変分原理版。

## PS 8.4 — 射影と最小二乗の幾何

**問題**: Ax = b の最小二乗解 x̂ = (AᵀA)⁻¹Aᵀb による射影行列 P = A(AᵀA)⁻¹Aᵀ が P² = P, Pᵀ = P を満たすことを示せ。

**回答**:

**P² = P**: P² = A(AᵀA)⁻¹Aᵀ · A(AᵀA)⁻¹Aᵀ = A(AᵀA)⁻¹(AᵀA)(AᵀA)⁻¹Aᵀ = A(AᵀA)⁻¹Aᵀ = P ✓

**Pᵀ = P**: Pᵀ = (A(AᵀA)⁻¹Aᵀ)ᵀ = A((AᵀA)⁻¹)ᵀAᵀ。(AᵀA)⁻¹ は対称なので = A(AᵀA)⁻¹Aᵀ = P ✓

**意味**: Pb は b の C(A) への直交射影。b − Pb = (I−P)b は C(A)⊥ = N(Aᵀ) の成分。

**核心**: 最小二乗 = 直交射影。この幾何学的視点が、正則化（射影先を変える）や制約付き最小化（射影の条件を変える）の理解に直結。

## PS 8.5 — 鞍点問題（KKT）

**問題**: min ½xᵀKx − fᵀx subject to Ax = b の KKT 条件を導き、鞍点行列を書け。

**回答**:

ラグランジュ関数: L(x, λ) = ½xᵀKx − fᵀx + λᵀ(Ax − b)。

KKT 条件（∂L/∂x = 0, ∂L/∂λ = 0）:

$$Kx + A^T\lambda = f$$
$$Ax = b$$

行列形式:

$$\begin{bmatrix} K & A^T \\ A & 0 \end{bmatrix}\begin{bmatrix} x \\ \lambda \end{bmatrix} = \begin{bmatrix} f \\ b \end{bmatrix}$$

この行列は**不定**（0 ブロックのため固有値が正負両方）。K が正定値でも全体は鞍点。

**物理的意味**: x は変位、λ はラグランジュ乗数（= 制約力）。Stokes 流れでは x = 速度、λ = 圧力。

**核心**: 制約付き最適化は鞍点問題に帰着する。解法には特殊な手法（Uzawa 法、ブロック前処理など）が必要。

## PS 8.6 — 線形計画と双対性

**問題**: 主問題 min cᵀx s.t. Ax = b, x ≥ 0 の双対問題を書き、弱双対性（双対の値 ≤ 主の値）を示せ。

**回答**:

**双対問題**: max bᵀy s.t. Aᵀy ≤ c。

**弱双対性の証明**: x が主問題の実行可能解、y が双対の実行可能解のとき:

$$c^Tx \geq (A^Ty)^Tx = y^T(Ax) = y^Tb = b^Ty$$

（第1の ≥ は Aᵀy ≤ c かつ x ≥ 0 による）

よって **bᵀy ≤ cᵀx**（双対の目的値 ≤ 主の目的値）。

**強双対性**: 両方が実行可能解を持つなら、最適解で bᵀy* = cᵀx*（ギャップ = 0）。

**核心**: 双対問題は「同じ最適値に別の空間から近づく」。主と双対を同時に解く内点法はこの構造を利用する。

## PS 8.7 — 随伴法（adjoint method）

**問題**: 目的関数 J(p) = gᵀu(p)（u は Ku = f(p) の解、p は設計変数ベクトル）の勾配 dJ/dp を随伴法で計算せよ。

**回答**:

**直接法**: 各 pᵢ に対して δuᵢ = K⁻¹(∂f/∂pᵢ) を解き、dJ/dpᵢ = gᵀδuᵢ。
→ 設計変数の数 m 回の求解が必要。O(m × solve)。

**随伴法**:
1. 順問題を解く: Ku = f
2. 随伴問題を解く: **Kᵀλ = g**（1回だけ）
3. 勾配: dJ/dpᵢ = λᵀ(∂f/∂pᵢ)

K が対称なら Kᵀ = K なので、随伴問題は順問題と同じ構造。

**コスト**: 順問題1回 + 随伴問題1回 = **計2回の求解で全勾配**。m に依存しない。

**導出**: J = gᵀu = gᵀK⁻¹f = (K⁻ᵀg)ᵀf = λᵀf。よって dJ/dp = λᵀ(∂f/∂p)。

**核心**: 随伴法は「微分を賢く計算する」技術。設計変数が多いほど（m >> 1）直接法に比べて圧倒的に有利。ニューラルネットの誤差逆伝播（backpropagation）は随伴法の離散版。
