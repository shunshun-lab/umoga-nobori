# Problem Set 回答集

> **Computational Science and Engineering** (Gilbert Strang) の各章 Problem Set から主要問題を選び、回答を整理したもの。
> 全問ではなく、各章の核心概念を確認できる問題を 5〜10 問ずつ選定。

## ファイル構成

| ファイル | 章 | 問題数 | テーマ |
|---|---|---|---|
| `ch1.md` | Ch1: Applied Linear Algebra | 10問 | 特殊行列、差分、境界条件、固有値、正定値、LU/QR/SVD |
| `ch2.md` | Ch2: A Framework for Applied Mathematics | 8問 | ばね系、A^TCA、最小二乗、グラフ/回路 |
| `ch3.md` | Ch3: Boundary Value Problems | 7問 | 微分方程式、弱形式、FEM、境界条件 |
| `ch4.md` | Ch4: Fourier Series and Integrals | 7問 | フーリエ係数、直交性、Gibbs、FFT |
| `ch5.md` | Ch5: Analytic Functions | 5問 | テイラー級数、特異点、留数、チェビシェフ |
| `ch6.md` | Ch6: Initial Value Problems | 7問 | 分散関係、安定性、増幅率G、CFL |
| `ch7.md` | Ch7: Solving Large Systems | 5問 | 帯幅、フィルイン、反復法、CG |
| `ch8.md` | Ch8: Optimization and Minimum Principles | 7問 | 最小二乗、正則化、KKT、随伴法 |

## 選定基準

1. 章の主張を直接確認する問題（概念理解）
2. 計算手順を身体化する問題（手を動かす）
3. 章をまたぐ接続が見える問題（横断理解）
