## PS 5.1 — テイラー級数と収束半径

**問題**: f(z) = 1/(1−z) のテイラー級数を z=0 で展開し、収束半径を求めよ。z=i で展開するとどうなるか。

**回答**:

**z=0 中心**: f(z) = 1 + z + z² + z³ + ... 。収束半径 R = 1（最も近い特異点 z=1 までの距離）。

**z=i 中心**: f(z) = 1/(1−z) = 1/((1−i)−(z−i))。

$$f(z) = \frac{1}{1-i}\sum_{n=0}^{\infty}\left(\frac{z-i}{1-i}\right)^n$$

収束半径 = |1−i| = √2（z=i から特異点 z=1 までの距離）。

**核心**: 収束半径は「最も近い特異点までの距離」で決まる。実軸上では見えない複素平面の特異点も制限になる。これがスペクトル法の精度を支配する。

## PS 5.1 — コーシーの積分定理

**問題**: ∮ z² dz を |z|=1 上で計算し、コーシーの定理を確認せよ。∮ 1/z dz も計算せよ。

**回答**:

**z² の積分**: z² は全平面で解析的。コーシーの定理より ∮ z² dz = 0。

直接計算: z = e^{iθ}, dz = ie^{iθ}dθ。
$$\oint z^2 dz = \int_0^{2\pi} e^{2i\theta} \cdot ie^{i\theta} d\theta = i\int_0^{2\pi} e^{3i\theta} d\theta = 0 \quad \checkmark$$

**1/z の積分**: z=0 に極がある → コーシーの定理は適用不可。

$$\oint \frac{1}{z}dz = \int_0^{2\pi} \frac{1}{e^{i\theta}}\cdot ie^{i\theta}d\theta = \int_0^{2\pi} i \, d\theta = 2\pi i$$

**核心**: 解析関数の積分は経路に依存しない（= 0）。特異点を囲むと留数が現れる。Res(1/z, 0) = 1 なので ∮ = 2πi · 1。

## PS 5.2 — 留数による実積分の計算

**問題**: ∫₀^∞ 1/(1+x²) dx を留数で計算せよ。

**回答**:

f(z) = 1/(1+z²) = 1/((z+i)(z−i))。上半面に極 z=i（1位）。

留数: Res(f, i) = 1/(i+i) = 1/(2i) = −i/2。

上半面の半円 C_R + 実軸で閉路:

$$\oint f(z)dz = 2\pi i \cdot \text{Res}(f, i) = 2\pi i \cdot \frac{1}{2i} = \pi$$

R→∞ で半円上の積分 → 0（|f| ≤ 1/R²）。

$$\int_{-\infty}^{\infty} \frac{dx}{1+x^2} = \pi \quad \Rightarrow \quad \int_0^{\infty} \frac{dx}{1+x^2} = \frac{\pi}{2}$$

検算: [arctan x]₀^∞ = π/2 ✓。

**核心**: 複素積分で実積分を「一瞬で」計算できる。留数は「特異点が積分にどれだけ寄与するか」を測る。

## PS 5.4 — チェビシェフ点と補間精度

**問題**: f(x) = 1/(1+25x²) を [-1,1] で等間隔点とチェビシェフ点で補間し、精度を比較せよ。

**回答**:

**等間隔点**: Runge 現象が発生。n が増えると端で振動が爆発する。n=10 で端の誤差 ≈ O(1)。

**チェビシェフ点**: xⱼ = cos(jπ/n)。端に点が集中し、Runge 現象を回避。n=10 で最大誤差 ≈ 10⁻²、n=20 で ≈ 10⁻⁵。

**理由**: f(z) = 1/(1+25z²) の特異点は z = ±i/5（虚軸上、実軸から距離 0.2）。

- 等間隔: 誤差 ∝ (ρ)^{−n} だが ρ が不利で、Lebesgue 定数が指数的に増大
- チェビシェフ: Lebesgue 定数 ∝ log n。誤差は ρ^{−n}（ρ = 特異点を含む楕円の半径）

**核心**: 「等間隔は危険、チェビシェフ点は安全」は複素平面の特異点で説明できる。補間点の分布が精度を決める。

## PS 5.4 — スペクトル法の指数精度

**問題**: スペクトル法（チェビシェフ展開）が有限差分法より速く収束する条件を述べよ。

**回答**:

**スペクトル法の誤差**: 解 u(x) が複素平面で解析的なら、チェビシェフ展開の誤差は

$$\|u - u_N\|_\infty \leq C \rho^{-N}$$

ρ > 1 は u(z) の特異点を含まない楕円の半径。ρ が大きいほど（特異点が遠いほど）指数的に速い。

**有限差分法の誤差**: O(hᵖ) = O(N⁻ᵖ)（p は差分の次数）。代数的収束。

**条件**: u が解析的なら、スペクトル法は **N を少し増やすだけで** 精度が指数的に向上する。u に不連続や角があると解析性が失われ、スペクトル法も代数的に劣化する。

| 解の性質 | スペクトル法 | 有限差分（2次） |
|---|---|---|
| 解析的 | O(ρ⁻ᴺ) 指数的 | O(N⁻²) |
| C^k | O(N⁻ᵏ) | O(N⁻²) |
| 不連続 | O(N⁻¹) + Gibbs | O(N⁻¹) |

**核心**: 「滑らかさ」ではなく「解析性」（複素平面での正則性）がスペクトル法の強さを決める。Ch4 のフーリエの教訓（滑らかさ→係数減衰）の精密化。
