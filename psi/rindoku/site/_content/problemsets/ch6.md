## PS 6.1 — 分散関係の導出

**問題**: 波動方程式 uₜ = cuₓ、熱方程式 uₜ = uₓₓ、分散方程式 uₜ = uₓₓₓ それぞれの分散関係 ω(k) を求め、性質を分類せよ。

**回答**:

u = e^{i(kx−ωt)} を代入:

| 方程式 | 代入結果 | ω(k) | 性質 |
|---|---|---|---|
| uₜ = cuₓ | −iω = ick | ω = ck | 保存（|e^{−iωt}| = 1）|
| uₜ = uₓₓ | −iω = −k² | ω = −ik² | 散逸（e^{−k²t} → 0）|
| uₜ = uₓₓₓ | −iω = −ik³ | ω = k³ | 分散（|e^{−ik³t}| = 1、速度 k² 依存）|

**移流**: 全周波数が同じ速度 c で移動。波形を保存。
**拡散**: 高周波ほど速く減衰。滑らかになる。
**分散**: エネルギーは保存するが、周波数ごとに速度が違い波形が崩れる。

**核心**: ω(k) の虚部が散逸（減衰/成長）、実部が伝播速度を決める。この分類が数値スキーム選択の出発点。

## PS 6.2 — 前進オイラーの安定性

**問題**: 熱方程式 uₜ = uₓₓ に前進オイラー＋中心差分を適用し、増幅率 G(k) を求めよ。安定条件を導け。

**回答**:

スキーム: U^{n+1}_j = U^n_j + R(U^n_{j+1} − 2U^n_j + U^n_{j−1})、R = Δt/(Δx)²。

U = G^n e^{ikjΔx} を代入:

$$G = 1 + R(e^{ikΔx} - 2 + e^{-ikΔx}) = 1 + 2R(\cos k\Delta x - 1)$$

cos kΔx − 1 ∈ [−2, 0] なので:

$$G \in [1 - 4R, \; 1]$$

安定条件 |G| ≤ 1:
- G ≤ 1: 常に成立
- G ≥ −1: 1 − 4R ≥ −1 → **R ≤ 1/2**

$$\boxed{\frac{\Delta t}{(\Delta x)^2} \leq \frac{1}{2}}$$

**核心**: 最も危険な波数は kΔx = π（最高周波数）。ここで G = 1−4R が最も −1 に近づく。Δx を半分にすると Δt は 1/4 にしなければならない。

## PS 6.2 — 後退オイラーの無条件安定性

**問題**: 後退オイラー（陰的）の増幅率を求め、無条件安定であることを示せ。

**回答**:

スキーム: U^{n+1}_j − R(U^{n+1}_{j+1} − 2U^{n+1}_j + U^{n+1}_{j−1}) = U^n_j。

U = G^n e^{ikjΔx} を代入:

$$G(1 - 2R(\cos k\Delta x - 1)) = 1$$

$$G = \frac{1}{1 + 2R(1 - \cos k\Delta x)}$$

分母 ≥ 1（R > 0, 1−cos ≥ 0）なので **|G| ≤ 1 は常に成立**。

代償: 毎ステップ連立方程式 (I − R·K)U^{n+1} = U^n を解く必要がある。行列は三重対角なので O(N) で解ける。

**核心**: 陰的スキームは Δt 制約を外す代わりに、各ステップで線形方程式を解く。大きな Δt が使えるなら総コストで有利。

## PS 6.3 — クランク・ニコルソン

**問題**: クランク・ニコルソンの増幅率を求め、2次精度であることを確認せよ。

**回答**:

スキーム（陽的と陰的の平均）:

$$U^{n+1} - \frac{R}{2}(\delta^2 U^{n+1}) = U^n + \frac{R}{2}(\delta^2 U^n)$$

増幅率:

$$G = \frac{1 + R(\cos k\Delta x - 1)}{1 - R(\cos k\Delta x - 1)} = \frac{1 - R(1 - \cos k\Delta x)}{1 + R(1 - \cos k\Delta x)}$$

分子 ≤ 分母（R > 0）なので **|G| ≤ 1 常に成立**（無条件安定）。

さらに: 真の G_exact = e^{−k²Δt}。テイラー展開で G_CN = 1 − k²Δt + (k²Δt)²/2 − ... = e^{−k²Δt} + O((k²Δt)³)。

時間で2次精度、空間で2次精度。

**核心**: 精度と安定のバランス。前進オイラー（1次・条件付き安定）→ 後退オイラー（1次・無条件安定）→ クランク・ニコルソン（2次・無条件安定）。

## PS 6.4 — 波動方程式とリープフロッグ

**問題**: uₜₜ = c²uₓₓ にリープフロッグスキームを適用し、CFL 条件 cΔt/Δx ≤ 1 を導け。

**回答**:

リープフロッグ: U^{n+1}_j − 2U^n_j + U^{n−1}_j = C²(U^n_{j+1} − 2U^n_j + U^n_{j−1})。

C = cΔt/Δx（クーラン数）。U = G^n e^{ikjΔx} を代入:

$$G + G^{-1} - 2 = C^2 \cdot 2(\cos k\Delta x - 1)$$

G + 1/G = 2 + 2C²(cos kΔx − 1) ≡ 2β。

$$G = \beta \pm \sqrt{\beta^2 - 1}$$

|G| = 1（安定）⇔ |β| ≤ 1 ⇔ β² ≤ 1。

最悪ケース kΔx = π: β = 1 − 2C²。|β| ≤ 1 → |1 − 2C²| ≤ 1 → **C ≤ 1**。

$$\boxed{c\frac{\Delta t}{\Delta x} \leq 1}$$

**核心**: CFL 条件は「数値の情報伝播速度 ≥ 物理の情報伝播速度」。波動方程式では特性線の速度 c がこの制約を決める。

## PS 6.5 — 移流拡散方程式

**問題**: uₜ + cuₓ = νuₓₓ で、ペクレ数 Pe = c·Δx/(2ν) > 1 のとき中心差分が不安定になる理由を説明せよ。

**回答**:

中心差分 + 前進オイラー:

$$U^{n+1}_j = U^n_j - \frac{c\Delta t}{2\Delta x}(U^n_{j+1} - U^n_{j-1}) + \frac{\nu\Delta t}{(\Delta x)^2}(U^n_{j+1} - 2U^n_j + U^n_{j-1})$$

G = 1 − ic(Δt/Δx)sin kΔx + 2R(cos kΔx − 1) で R = νΔt/(Δx)²。

問題: cΔx/(2ν) > 1 のとき、移流項が支配し、拡散による安定化が不足。高周波の G が実軸負方向に |G|>1 になりうる。

**対策**:
1. **風上差分**: 移流方向にバイアスした差分（人工粘性を追加）
2. **メッシュを細かく**: Pe < 1 になるまで Δx を小さくする
3. **陰的スキーム**: 時間方向の安定性を確保

**核心**: 対流支配 (Pe >> 1) は数値的に「悪い」。物理の拡散が数値の振動を抑えられないため、人工的な安定化が必要。

## PS 6.1 — Duhamel の原理

**問題**: uₜ = Lu + f(x,t), u(x,0) = 0 の解が Duhamel の公式で与えられることを確認せよ。

**回答**:

同次問題 vₜ = Lv, v(x,0) = g(x) の解を v(x,t) = e^{Lt}g(x) と書く。

Duhamel の公式:

$$u(x,T) = \int_0^T e^{L(T-t)}f(x,t)\,dt$$

意味: 時刻 t に入ったソース f(x,t) は、残り時間 T−t だけ「成長/減衰」する。全ソースの寄与を積分。

**離散版**: U_N = S^N U_0 + Σ S^{N-n} F_n。S は1ステップの発展行列。

安定性の意味: |S の固有値| ≤ 1 なら、各 F_n の寄与が時間とともに増幅されない。

**核心**: Duhamel はソース項（外力/誤差/入力）が時間発展でどう増幅されるかを見る道具。デバッグでは「この誤差はいつ入って、どれだけ成長したか」を特定できる。
