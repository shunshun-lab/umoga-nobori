## 解説メモ

### §1.4 逆行列とデルタ関数の構造

**核心**: 逆行列の列 = 点荷重への応答 = Green関数。

**一様荷重 vs 点荷重**:

| | 一様 $f=1$ | 点荷重 $f=\delta(x-a)$ |
|---|---|---|
| 力のかかり方 | 全体に均等 | 1点に集中 |
| $u''$ | 至るところ定数 | 荷重点以外でゼロ |
| $u$ の形 | 放物線（滑らか） | ランプ（折れ線） |
| イメージ | 梁に雪が均一に積もる | 梁の1点を指で押す |

なぜ形が違うか：$u'' = $ 曲率。一様荷重 → 曲率一定 → 滑らか。点荷重 → 荷重点以外は曲率ゼロ（直線）、荷重点だけ角。

**Green関数（固定-固定）**: $-u'' = \delta(x-a)$, $u(0)=u(1)=0$

$$u(x) = \begin{cases}(1-a)x & x \leq a\\(1-x)a & x \geq a\end{cases}$$

**$K_n^{-1}$ の公式**（$i \leq j$）: $(K_n^{-1})_{ij} = \frac{i(n+1-j)}{n+1}$

対角まで上がって下がるランプ = 連続版 $G(x,a)$ の離散版。

**任意の $f$ = 点荷重の重ね合わせ**:
- 離散：$u = K^{-1}f = \sum f_j(\text{第}j\text{列})$
- 連続：$u(x) = \int f(a)G(x,a)\,da$

**Woodbury-Sherman-Morrison**: $K = T - uv^T$ → $K^{-1} = T^{-1} + \frac{T^{-1}uv^TT^{-1}}{1 - v^TT^{-1}u}$

### §1.7 数値的に安定な分解（LU / QR / SVD）

**安定性の源 = 直交行列 $Q$**：$\|Qx\| = \|x\|$ → 誤差を増幅しない。

| | $A = LU$ | $A = QR$ | $A = U\Sigma V^T$ |
|---|---|---|---|
| 構造 | 下三角 $\times$ 上三角 | 直交 $\times$ 上三角 | 直交 $\times$ 対角 $\times$ 直交 |
| 安定性 | 低い（小ピボットで危険） | 高い（$Q$ が長さを保つ） | 最高（両側直交） |
| 速度 | $\frac{2}{3}n^3$ | $\frac{4}{3}n^3$ | 遅い |
| 用途 | $Ax=b$（正方） | 最小二乗、長方行列 | ランク判定、条件数、PCA |

**LUが危ない理由**: ピボットが小さい → 乗数 $\ell = \text{係数}/\text{ピボット}$ が巨大 → 誤差が増幅。対策 = 部分ピボッティング $PA = LU$。正定値ならピボット常に正 → 安全。

**QR の2つの方法**:

| | Gram-Schmidt | Householder |
|---|---|---|
| やること | 射影を引く | 反射で消す |
| 安定性 | やや不安定 | 安定（標準） |

**SVD の幾何的意味**: $A$ の作用 = $V$ で回転 → $\Sigma$ で伸縮 → $U$ で回転。

**条件数**: $c(A) = \sigma_{\max}/\sigma_{\min}$。$K_n$ では $c \approx \frac{4}{\pi^2}(n+1)^2$ → メッシュを細かくすると $O(n^2)$ で悪化。

### 正定値は対称が前提

$u^TAu$ を展開すると、非対称部分（歪対称 $\frac{A-A^T}{2}$）は $u^T(\text{歪対称})u = 0$ で消える。二次形式は対称部分 $\frac{A+A^T}{2}$ だけで決まるので、最初から対称に限定しても一般性を失わない。

対称であることの恩恵：

| 性質 | 対称なら | 非対称なら |
|---|---|---|
| 固有値 | すべて実数 | 複素数もあり得る |
| 固有ベクトル | 直交（$Q\Lambda Q^T$） | 直交とは限らない |
| $LDL^T$ 分解 | 使える | 使えない |

### 上三角差分行列 $U$ と和の行列 $U^{-1}$

$T_n$（自由-固定）のピボットが全部1 → 消去で出る $U$ がきれいな差分行列：

$$U = \begin{bmatrix}1&-1&0\\0&1&-1\\0&0&1\end{bmatrix} \quad U^{-1} = \begin{bmatrix}1&1&1\\0&1&1\\0&0&1\end{bmatrix}$$

| 行列 | 操作 | 例 |
|---|---|---|
| $U$ | 隣との差を取る | $[5,3,1] \to [2,2,1]$ |
| $U^{-1}$ | 下から累積和を取る | $[2,2,1] \to [5,3,1]$ |

差分の逆 = 和。微分の逆 = 積分の離散版。

$T_n = U^TU$（対称だから $L = U^T$）。$T_n^{-1} = U^{-1}(U^T)^{-1}$ → 和の行列の積 → 成分が定数パターンになる理由。

## 引っかかったところ

### 1. 一様荷重 vs 点荷重

| 疑問 | 解消 |
|---|---|
| 「放物線」と「ランプ」の違いの意味 | $u'' = $ 曲率。一様荷重なら曲率一定で滑らか、点荷重なら荷重点以外は曲率ゼロで直線、荷重点で折れる |

### 2. 正定値の前提

| 疑問 | 解消 |
|---|---|
| 正定値は対称が前提？ | はい。$u^TAu$ の歪対称部分は $u^T(\text{歪対称})u = 0$ で消えるので、対称部分だけで決まる |

### 3. 差分行列 $U$ と和の行列 $U^{-1}$

| 疑問 | 解消 |
|---|---|
| ノートの「$U$ が差分、$U^{-1}$ が和」とは？ | $U[5,3,1]=[2,2,1]$（差を取る）、$U^{-1}[2,2,1]=[5,3,1]$（累積和で戻す）。微分↔積分の離散版 |
| $T_n^{-1}$ の成分がきれいなのはなぜ | $T_n = U^TU$ → $T_n^{-1} = U^{-1}(U^T)^{-1}$ = 和の行列の積 |

### 4. 数値的安定性

| 疑問 | 解消 |
|---|---|
| 「安定」とは | 丸め誤差が増幅されないこと。直交行列 $Q$ は $\|Qx\|=\|x\|$ で長さを保つから安定 |
| LUが危ないのはなぜ | 小ピボット → 巨大な乗数 → 誤差が増幅。正定値なら安全 |
| QR vs SVD の使い分け | QR: 最小二乗・長方行列。SVD: ランク判定・条件数。速度と情報量のトレードオフ |
