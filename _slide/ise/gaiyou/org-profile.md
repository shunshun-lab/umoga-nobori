## 1. MVV（Mission / Vision / Value）

### Mission（使命 / なぜ存在するか）  🔶要HP確認 / 💭たたき台
- 伊勢の地から、日本の伝統文化を次の世代へ継承する
- 若い世代・子どもたちの健やかな育ちを支える
- 世代をこえた温かなつながりをつくる

### Vision（目指す姿 / どんな未来を）  💭たたき台
- 「変わらないために、変わり続ける」— 伊勢の式年遷宮＝「常若（とこわか）」の思想を、現代人の生き方の軸に
- 進む道は分かれても、伊勢でつながった心がずっとひとつに在り続けるコミュニティ

### Value（大切にする価値 / 5つのこころ）  ✅既存素材
- **奉仕** — 人の役に立つ喜び
- **友愛** — 世代をこえた友情
- **責任** — 自分の人生に主体的であること
- **忍耐** — 続ける力・待つ力
- **進取** — 一歩踏み出す勇気
- ※スライドでは「身につけるもの」でなく「関わるうちに自然と育つもの」として提示

## 3. 各論ごとの実績（これまで）  ✅HP past-events / annual-plan で確認

### 朔日会（ついたちかい）@東京
- **毎月2日**、東京都内で定例開催（年間事業）✅
- 月替わりで伊勢の**朔日餅（全11種類）**を味わう。例: かしわ餅(4種目)→麦手餅(5)→笹わらび餅(6)→八朔粟餅(7)→萩の餅(8/年1回限定) ✅
- 5月は深夜0:30開始の「朔日詣り」回など、回ごとに趣向あり ✅
- 累計開催回数・延べ参加者数は非公開 🔶（HPに数値記載なし）

### リアル伊勢 / 伊勢ツアー（おかげ横丁参拝）
- **新年**(1/31-2/1)・**春**(2/28-3/1)・**秋**(10/31-11/1) におかげ横丁で参拝 ✅
- 宿泊予約ありの「伊勢ツアー 朔日参り」（3/31-4/1、8/31-9/1）✅

### 京都イベント
- **4月: 都をどり鑑賞＆祇園・花見小路のお茶屋で舞妓遊び体験（part1）** ✅（実施実績として past-events に記載）

### お木曳き奉仕
- **2026年 5/17・8/2** に奉仕枠を募集（**成人10名・学生10名の少人数制**）✅
- ※お木曳き自体は2033年遷宮へ向けた伊勢全体の伝統行事。前回2006年は延べ約23万人参加（伊勢全体の数字であり常磐会の実績ではない）

### 夏至祭
- **2026年 6/20-21**、二見興玉神社で実施予定 ✅

### 伊勢神宮基礎講座シリーズ（学びコンテンツ）
- vol1「歴史・神話・常若」/ vol2「参拝と歩き方」/ vol3「お木曳き」/ vol4「おかげ参りと文化」を制作 ✅（自主制作物）
- ※2027年2/2に「伊勢神宮100人講演会」開催に向けた決起集会を予定（青山）✅

### 規模に関する数値
- 会員数・延べ参加者数などの実績数値は **HP非公開**（資料で謳う場合は別途ヒアリングが必要）🔶
