# NotebookLM 指示書 — 琴平 空き家演劇祭 ワンページ説明資料

このファイルは、NotebookLM に「琴平 空き家演劇祭」の**1ページ説明資料**を作らせるための指示一式。

---

## 0. 使い方（手順）

1. NotebookLM で新規ノートブックを作る。
2. ソースとして **`akiya-engekisai.md`** をアップロード（このファイルと同じ `kotohira/docs/` にある企画書本文）。
3. 下の【プロンプト】をそのまま貼って生成させる。
4. 出力を見て、足りない／盛りすぎなら【調整指示の例】を追って投げる。

> ソースは1ファイルだけにすると、NotebookLM が他資料に引っ張られず精度が上がる。

---

## 1. 生成物の定義（ゴール）

- **A4・1ページに収まる**、初見の人が30秒で全体像をつかめる説明資料。
- 読み手：**助成金審査員・地域の協賛候補（バス会社／信金）・行政・参加検討アーティスト**。
- トーンは「叩き台だが、骨格は固い」。誇張せず、未確定は未確定と書く。

---

## 2. NotebookLM に貼るプロンプト

```
あなたは芸術祭の企画書を、関係者向けの「1ページ説明資料」に要約する編集者です。
アップロードしたソース（琴平 空き家演劇祭 企画書 DRAFT v0.2）だけを根拠に、
A4・1ページに収まる説明資料を日本語で作ってください。推測や脚色はせず、
ソースにない数字・団体名は足さないこと。

# 構成（この順・この見出しで）

2. 基本情報：開催地／時期／中核体制（AD・伴走・財務）を1〜2行で
3. コンセプト：空き家を小劇場に、パックチケットで回遊、町を舞台化（3〜4行）
4. なぜやるか（課題）：滞在1〜2時間・回遊性の弱さ、体験コンテンツ不足（箇条書き3点）

8. 期待効果：滞在延伸／関係人口／空き家再評価、中期はこんぴら歌舞伎と「演劇月間」化

# 体裁ルール

- 各セクションは見出し＋短文／箇条書き。読み手が審査員・協賛候補・行政だと意識する。
- 末尾に必ず「※ DRAFT v0.2 / 2026-06-04 / 名称を挙げた団体・財団からの正式同意は未取得」を入れる。
```

## 3. スライド（音声/動画）にする場合の追加指示

NotebookLM の「音声概要」や「スライド／動画」機能で説明資料化するなら、上に続けて：

```
この内容を、初見の人に説明する1本のスライド（または音声概要）にしてください。
・尺は2〜3分相当。
・最初の15秒で「参拝で終わる町を、回遊・滞在する町へ」という狙いを言い切る。
・数字は「助成4本410万円」「動員3,000人」「純利益100〜300万円」の3つだけを強調。
・最後に「決定的な3要因（助成申請・協賛内諾・アーティスト確定）」で締める。
・未確定事項は「現時点では未内諾」と正直に触れる。
```

---

## 4. 調整指示の例（出力を直すとき）

- 長すぎる →「全体を700字以内に。コンセプトと数字を残し、背景は2行に削って。」
- 数字が散漫 →「強調する数字を 410万円・3,000人・100〜300万円 の3つに絞って。」
- 盛りすぎ →「ソースにない表現（『日本初』等）を削除。トーンは『叩き台』に戻して。」
- 審査員向けに寄せる →「助成金審査員が読む前提で、社会的意義（関係人口・空き家活用）を先頭に。」
