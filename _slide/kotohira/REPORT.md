**調査日**: 2026-03-15
**URL**: https://kotohira.civicship.app/community/kotohira

## 1. サービス概要

「琴平デジタル町民」は、香川県琴平町の地域活性化を目的としたポイント型シビックシップ・プラットフォーム。住民や関係人口が「お手伝い（クエスト）」でポイントを貯め、地域の「体験（アクティビティ）」にポイントで参加できる仕組み。

- **プラットフォーム**: civicship.app（共通基盤）
- **コミュニティ名**: kotohira
- **運営拠点**: コトリ コワーキング＆ホステル琴平（香川県仲多度郡琴平町720-15）

## 2. サイト構成・画面一覧

### 2.1 トップ（Opportunities）ページ
- **URL**: `/community/kotohira/opportunities`
- ヘッダー: ロゴ + 検索バー（「タップして検索する」）
- **もうすぐ開催予定**: 直近の体験・クエストを横スクロールで表示
- **CTA バナー**: 「ポイントを獲得すると、無料で体験に参加できます。ポイントがもらえるお手伝いを探してみましょう」
- **すべての体験**: 全 Opportunity カードを一覧表示
- **ボトムナビ**: 「見つける」「マイページ」の2タブ

### 2.2 体験詳細ページ（つかう / Activity）
- ヒーロー画像
- タイトル（例:「【つかう】琴平mobi乗車」）
- 場所情報
- **体験できること**: テキスト説明 +「もっと見る」展開
- **必要ポイント**: 例「1,000pt必要」
- **日付を選択** ボタン（CTA）
- **依頼人**: 「琴平デジタル町民さん」
- **集合場所**: 住所 + Google Maps 埋め込み
- **開催日**: 日程カード（日付・時間・ポイント・予約状態）
- **注意事項**: 4項目の共通ルール

### 2.3 クエスト詳細ページ（ためる / Quest）
- ヒーロー画像（ポスター風ビジュアル：内容・日時・場所・ポイント・定員を記載）
- タイトル（例:「【ためる】3/19(木)～21(土) 讃岐おでん店頭のお手伝い」）
- 場所情報
- **助けてもらいたいこと**: 依頼内容の説明
- **獲得ポイント**: 例「3,000ptもらえる」
- **日付を選択** ボタン（CTA）
- **依頼人**: 「琴平デジタル町民さん」
- **集合場所**: 住所 + Google Maps
- **開催日**: 日程カード（日付・時間・ポイント・満席状態）
- **注意事項**: Activity と同じ共通ルール

### 2.4 ログイン / 会員登録ページ
- **URL**: `/community/kotohira/login`
- LINEログインのみ（ボタン1つ）
- 利用規約 + プライバシーポリシーへの同意チェックボックス

### 2.5 利用規約ページ
- **URL**: `/community/kotohira/terms`
- 全11条（事前予約、自己責任、主催者の指示、法令遵守、料金、中止・変更、服装・持ち物、飲食、キャンセル、web会員、その他）

### 2.6 プライバシーポリシーページ
- **URL**: `/community/kotohira/privacy`

## 4. UX / 機能面の特徴

| 項目 | 内容 |
|------|------|
| **認証** | LINE ログインのみ（ソーシャルログイン一本化） |
| **ポイントシステム** | クエスト参加でポイント獲得 → 体験でポイント消費 |
| **予約フロー** | 日付選択 → 主催者承認後に確定 |
| **予約締切** | 各日程の2日前まで受付 |
| **キャンセル** | 開催日の前日まで可能 |
| **地図** | Google Maps 埋め込み（集合場所表示） |
| **ナビゲーション** | ボトムタブ2つのシンプル構成 |
| **検索** | ヘッダーに検索バー |
| **レスポンシブ** | モバイルファーストUI |

## 5. コンテンツの傾向

- **讃岐おでん関連が圧倒的に多い** — 13件中8件がおでん関連（店頭手伝い、仕込み、買い出し、味見）
- **交通系体験が充実** — mobi（地域交通）、高速バス（関西往復）
- **宿泊体験** — コトリ琴平での宿泊がポイントで利用可能
- **ユニーク体験** — わんこうどんのお手伝い
- **運営主体が一元化** — 全て「琴平デジタル町民さん」（＝コトリ琴平）が依頼人

## 6. 技術スタック（観察ベース）

- **フレームワーク**: Next.js（RSC / `_next/static/chunks`）
- **ホスティング**: Vercel推定（civicship.app ドメイン）
- **地図**: Google Maps JavaScript API
- **認証**: LINE Login
- **状態管理**: Apollo Client（DevTools コンソールメッセージ確認）
- **画像最適化**: Next.js Image（`/_next/image` エンドポイント）

## 7. 気になった点 / 改善候補

1. **画像読み込みエラー**: 複数の Opportunity 画像で404エラー発生（webp, jpg, PNG）
2. **終了済みクエストの表示**: 2月開催済みのクエストがまだ一覧に表示されている（アーカイブ処理なし）
3. **体験カテゴリが2つのみ**: 「つかう」「ためる」以外のカテゴリ拡張余地あり
4. **依頼人の多様性**: 現状すべて「琴平デジタル町民さん」— 個人・店舗からのクエスト投稿が増えると活性化しそう
5. **ポイント獲得手段の偏り**: おでん関連に集中しており、多様な参加導線があるとよい
6. **満席表示**: 一部日程が「満席」「予約受付終了」になっており、需要はある模様
