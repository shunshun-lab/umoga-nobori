# 納品・権限移譲ガイド

本システムは Google Sheets API を使用して割引ルールを管理しています。
クライアント（納品先）へシステムを引き渡す際、以下の手順で権限設定や移譲を行うことを推奨します。

## 1. Google Sheets の共有設定（必須）

割引ルールを管理するスプレッドシートの編集権限をクライアントに付与します。

1. **対象シート**: [Nobori Pricing Rules](https://docs.google.com/spreadsheets/d/1vVLc_q4F4i186AuzSo82N-9VBs1rQt3DQKRtFiM7r-8/edit?usp=sharing)
2. **手順**:
   - シート右上の「共有」ボタンをクリック
   - クライアントのGoogleアカウント（Gmailアドレス等）を入力
   - 権限を「編集者」または「オーナー」に設定して送信
   - **完了**: これでクライアント自身がブラウザで割引ルールを編集できるようになります。

## 2. システム連携用アカウント（Service Account）の扱い

システムがシートを読み込むために使用している「Service Account（ロボット用アカウント）」の扱いについて、2つのパターンがあります。

### パターンA：現在の設定をそのまま引き渡す（簡易版）
現在設定されている `GOOGLE_SERVICE_ACCOUNT_EMAIL` と `GOOGLE_PRIVATE_KEY` をそのまま利用し続けます。
**条件**: 開発者（あなた）が作成したGoogle Cloudプロジェクトを今後も維持・管理する場合。

- **メリット**: 何も設定変更が不要。
- **デメリット**: 開発者がプロジェクトを削除するとシステムが停止する。

### パターンB：クライアント専用のアカウントを作成する（推奨）
クライアント自身のGoogle環境でService Accountを発行してもらい、設定を切り替えます。
将来的にも安定して運用するために、こちらを推奨します。

**移行手順**:

1. **Google Cloud Projectの作成**
   - クライアントに [Google Cloud Console](https://console.cloud.google.com/) にアクセスしてもらう。
   - Googleアカウントでログインし、「新しいプロジェクト」を作成。

2. **APIの有効化**
   - 作成したプロジェクトで「APIとサービス」→「ライブラリ」へ移動。
   - "Google Sheets API" を検索し、「有効にする」をクリック。

3. **Service Accountの作成**
   - 「IAMと管理」→「サービスアカウント」へ移動。
   - 「サービスアカウントを作成」をクリック（名前は `nobori-app-sync` など任意）。
   - 役割は特に設定不要（「続行」→「完了」）。

4. **キーの発行**
   - 作成されたサービスアカウントをクリックし、「キー」タブへ。
   - 「鍵を追加」→「新しい鍵を作成」→「JSON」を選択して作成。
   - JSONファイルがダウンロードされます。**これはパスワード同様に重要です。**

5. **シートへの招待**
   - 作成したサービスアカウントのメールアドレ（例: `nobori-app-sync@project-id.iam.gserviceaccount.com`）をコピー。
   - スプレッドシートの「共有」設定で、このメールアドレスを「閲覧者（または編集者）」として追加。
     - ※システムは読み取りのみ行う場合は「閲覧者」でOKですが、将来書き込みも行うなら「編集者」にします。

6. **環境変数の更新**
   - Vercelなどのホスティング設定画面（Environment Variables）で以下を更新します。
     - `GOOGLE_SERVICE_ACCOUNT_EMAIL`: 新しいサービスアカウントのメールアドレス
     - `GOOGLE_PRIVATE_KEY`: ダウンロードしたJSON内の `private_key` の値（`-----BEGIN PRIVATE KEY...` から始まる長い文字列）

## 3. 運用・編集方法

クライアントへの説明用です。

- **割引ルールの変更**:
  - 共有されたスプレッドシートを直接編集してください。
  - 行を追加・削除したり、割引率（Rate）を変更できます。
- **システムへの反映**:
  - アプリ管理画面（`/admin` 等、実装箇所）にある「Google Sheets 連携」タブを開きます。
  - 「同期」ボタンを押すと、即座にシートの内容がシステムに取り込まれます。
  - **注意**: シートを編集しただけでは反映されません。必ず「同期」ボタンを押してください。
