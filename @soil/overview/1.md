# PRD: のぼり専門店 オンライン見積もりシステム

## 1. プロダクト概要
ユーザーがオンラインでオリジナルの「のぼり」や「幕」を注文できるWebアプリケーション。直感的なUIでサイズ、生地、オプションを選択し、リアルタイムで見積もり価格を確認しながら注文に進むことができる。

## 2. ターゲットユーザー
- オリジナルのぼりを作成したい店舗オーナー、イベント主催者
- デザインデータ（Illustrator/Photoshop）を入稿可能なデザイナー
- 小ロットから大ロットまで発注したい法人・個人

## 3. 主要機能要件

### 3.1. 見積もりシミュレーション
ユーザーは以下の項目を選択し、即座に価格を確認できる。
- **サイズ選択**:
  - レギュラー (60×180cm)
  - スリム (45×180cm)
  - ショート (60×150cm)
  - ミニ (45×150cm)
- **生地選択**:
  - ポンジ (標準)
  - トロマット (厚手・丈夫)
  - ツイル (高級感)
- **枚数入力**:
  - 1枚から注文可能
  - **数量割引**: 10枚以上で10%OFF、30枚以上で15%OFF、50枚以上で20%OFF、100枚以上で25%OFF
- **オプション**:
  - 四方三巻縫製 (補強)
  - 防炎加工 (商業施設向け)
  - 棒袋加工
  - スティック他付属品

### 3.2. デザイン入稿機能
- ドラッグ＆ドロップによるファイルアップロード
- 対応フォーマット: AI, PDF, PSD, JPG, PNG
- 入稿なし（後日入稿など）のオプション対応

### 3.3. カート・決済連携 (Shopify)
- 見積もり内容に基づいた商品をShopifyのカートに追加
- Shopifyのチェックアウトプロセスを利用して決済・配送手続き
- Storefront APIを利用したシームレスな連携

## 4. 技術スタック

### フロントエンド
- **Framework**: React 19 + Vite 7
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **State Management**: Zustand 5
- **HTTP Client**: Axios
- **UI Components**: React Dropzone (アップロード)

### バックエンド / インフラ
- **Platform**: Shopify (Storefront API)
- **Hosting**: Vercel (推奨)

## 6. 非機能要件
- **レスポンシブデザイン**: スマートフォン、タブレット、PCでの快適な操作
- **パフォーマンス**: リアルタイム価格更新の遅延なし
- **拡張性**: 将来的な商品追加（横断幕、タペストリーなど）を考慮したコンポーネント設計

## 7. 画面遷移フロー
1. **トップページ (見積もりフォーム)**
2. **条件選択 (サイズ・生地・枚数・オプション)**
3. **デザインデータアップロード**
4. **「カートに追加」押下**
5. **Shopify チェックアウト画面 (別タブまたはリダイレクト)**
